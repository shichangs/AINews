## 1. 旗舰模型与 AI 产品

### 🔥 GPT-5.6 Sol、Terra、Luna 全系 GA，安全拦截量升至前代 10 倍

GPT-5.6 Sol 上线即报 Agents' Last Exam **53.6** 分，OpenAI 称领先 Fable 5 13.1 分、成本约其 1/4。三档 7/9 起在 ChatGPT、Codex 与 API 全面 GA，24 小时内完成全球滚动（获批过程与定价详见 07-09 归档），新增 `max` 与 `ultra` 档——后者默认 4 个 agent 并行作业。自选基准另含 Coding Agent Index **80**（Fable 5 为 77.2）、BrowseComp 92.2%（ultra）、OSWorld 2.0 62.6%；但同页表格显示 SWE-bench Pro 上 Sol 64.6% 低于 Fable 5 的 80%——口径均由 OpenAI 挑选，待第三方复测。安全侧动作同样密集：cyber safeguards 拦截量为前代约 **10 倍**，发布前投入约 700,000 A100e GPU 时做自动化红队；Trusted Access for Cyber 用户须 **9/1** 前启用硬件 passkey，否则回落默认权限。Altman 当日对 Bloomberg 称公司在与美国政府磋商中"做了很多修改"。Grok 4.5 同日公开上线（详见 07-09 归档）。

（来源：[OpenAI 官方](https://openai.com/index/gpt-5-6/) · [Bloomberg](https://www.bloomberg.com/news/articles/2026-07-09/altman-says-openai-made-many-changes-during-talks-with-us) · [Engadget](https://www.engadget.com/2210308/openai-rolls-out-gpt5-6-july-9/)）

### 🔥 OpenAI 发布 ChatGPT Work，Codex App 并入桌面端、Atlas 启动退役

Codex 每周 **500 万**用户的独立 App 即日并入新 ChatGPT 桌面端，Atlas 独立浏览器同步启动退役。ChatGPT Work 是 OpenAI 7/9 推出的 agent：跨应用与文件取材、可连续数小时拆解并推进复杂项目，产出表格、幻灯片、文档与网页应用，由同日 GA 的 GPT-5.6 驱动。上线节奏：网页与移动端先开 Pro / Enterprise / Edu，Plus / Business 数日内跟进；桌面端全档（含免费）即日全球可下载。配套发布 Sites 公测（把工作成果变成可分享的交互网站）、桌面 Computer Use 与内置浏览器；Chrome 扩展改为侧栏形态，Atlas 用户将收到迁移指引，旧版桌面应用更名 ChatGPT Classic。Codex 用户中已有超 **100 万**人将其用于非开发工作，OpenAI 称这是 Work 面向全职能扩张的基础。下一节点：Plus / Business 档位数日内开通，企业 admin 可在 Admin Console 设置用量上限。

（来源：[OpenAI 官方](https://openai.com/index/chatgpt-for-your-most-ambitious-work/) · [US News（Reuters）](https://money.usnews.com/investing/news/articles/2026-07-09/openai-launches-chatgpt-work)）

### 🔥 Meta 发布 Muse Spark 1.1，Meta Model API 公测首开

开发者首次可直接调用 Muse 系模型——Meta Model API 7/9 公测，Muse Spark 1.1 携 **1M** token 上下文入场。这是 Meta Superintelligence Labs 本周第二发（7/7 Muse Image / Video 详见 07-08 归档）：多模态推理 agentic 模型，作为主 agent 可规划任务并向并行 subagent 分派执行，1M 上下文支持主动管理（记忆动作、回溯早期信息、压缩时保留关键步骤）；计算机使用场景中被训练为"自动化更快时写脚本、直接交互更简单时点击"。首批 API 伙伴为 Replit、Cline 与 Box；模型同日在 Meta AI app 与 meta.ai 的 Thinking 模式可用。时点耐人寻味：GPT-5.6 与 Grok 4.5 同日公开上线，三家模型 24 小时内同时向开发者开闸。观察点：Meta 首次补上"模型—API"商业化闭环后，能否从 OpenAI / Anthropic 的 API 存量盘里切走 agentic 工作负载。

（来源：[Meta AI 官方](https://ai.meta.com/blog/introducing-muse-spark-meta-model-api/) · [Meta 开发者博客](https://developer.meta.com/ai/resources/blog/build-with-muse-spark)）
