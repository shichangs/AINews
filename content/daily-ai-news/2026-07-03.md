## 4. AI Agent 与企业部署

### Cognition 推 Devin Security Swarm，跨全库找漏洞

50 个真实 GHSA 漏洞测得 **72%** 召回、单次约 **$90**。**Cognition** 7/1 发布 **Devin Security Swarm**，用 "Agentic MapReduce" 架构扫描大型代码库、在沙箱验证可利用性并自动开修复 PR。官方 eval 覆盖 14 种语言的 **50** 个真实 GHSA 漏洞，报告 **72%** 召回率、单次运行约 **$90**，并称抓到其他工具漏掉的严重漏洞。下一检验点：把安全扫描做成 agent 集群、成本低于以往 agentic 方案后，企业能否接受其误报率与自动 PR 质量，是采用率的首个硬信号。

（来源：[Cognition](https://cognition.com/blog/introducing-devin-security-swarm) · [Devin Security Swarm Eval](https://devin.ai/blog/security-swarm-eval)）

### CAIS / Scale 远程劳动指数：Fable 5 居公开模型首位（16.1%）

公开模型在真实远程外包上仅 **16.1%** 达可交付标准。**CAIS** 与 **Scale AI** 7/1 更新 Remote Labor Index，**Claude Fable 5** 以 **16.1%** 居公开模型首位；测评基于 23 个领域的 **240** 个真实远程外包项目，以"客户是否会接受为真实交付物"为通过标准。下一检验点：即便榜首模型也仅约六分之一项目达可交付标准，说明 agent 距端到端替代远程工时仍远；后续版本纳入更多闭源模型是关键跟踪点。
