## 1. 主要 AI Labs 动态

### 🔥 Anthropic 机密递交 S-1，或成史上前三大 IPO

若以 **$1 万亿** 估值挂牌，将跻身史上前三大 IPO（仅次于 SpaceX、沙特阿美）。Anthropic **6/1** 宣布已向 SEC 机密递交 S-1 草案，发行股数与价格尚未确定，公司称"待 SEC 审完后即有上市选项"。递交早于此前今秋时间表——数日前 Anthropic 刚完成 **$65B** H 轮、投后估值 **$965B**；年化营收已约 **$47B**（一年前约 $10B）。抢跑的意义在排序：分析师多认为先上市者在这轮 AI 募资竞赛中表现更好（参照 2019 年 Lyft 先于 Uber 上市、首日表现更佳）。OpenAI 尚未递交，两家此前都预计今秋挂牌。下一关键节点：**6/12** SpaceX 以代码 SPCX 在 Nasdaq 上市，为这批万亿级 AI IPO 设定价锚；OpenAI 是否在数周内跟进是第二观察点。

（来源：[NBC News](https://www.nbcnews.com/business/corporations/anthropic-files-ipo-openai-rcna347897) · [CNBC](https://www.cnbc.com/2026/06/01/anthropic-ipo-s1-prospectus.html) · [Fortune](https://fortune.com/2026/06/01/anthropic-s1-confidential/)）

*披露：本报告由 Claude 撰写，Anthropic 为 Claude 开发方。IPO 递交以 NBC、CNBC、Fortune、NPR 等非 Anthropic 来源交叉确认。*

### OpenAI 前沿模型与 Codex 上线 AWS Bedrock

GPT-5.5 与 Codex 上线 **AWS Bedrock**，分发不再独靠 Azure。OpenAI **6/1** 宣布 **GPT-5.5**、**GPT-5.4** 与 **Codex** 在 Amazon Bedrock 全量可用（4 月起为有限预览），定价对齐 OpenAI 一方价、Codex 按 token 计费；区域上 GPT-5.5 落俄亥俄、GPT-5.4 落俄亥俄与俄勒冈。OpenAI 称每周已有逾 **500 万**人用 Codex 写、改、调、测代码。此举把 OpenAI 模型铺上最大公有云 AWS，与长期主力 Bedrock 的 Anthropic 争夺企业分发——这条线源于 OpenAI 与 Microsoft **4/27** 重组、终止微软独家。下一关键节点：更多模型与区域将陆续上架，企业可用既有 AWS 安全与采购治理把前沿模型推上生产。

（来源：[OpenAI](https://openai.com/index/openai-frontier-models-and-codex-are-now-available-on-aws/) · [AWS News Blog](https://aws.amazon.com/blogs/aws/get-started-with-openai-gpt-5-5-gpt-5-4-models-and-codex-on-amazon-bedrock/)）

## 2. 芯片与半导体

### 🔥 NVIDIA Vera Rubin 进入全量产，今秋出首批货

全球 **150** 家台厂、**350+** 工厂、30 国已在量产爬坡，今秋出首批货。NVIDIA **6/1** GTC Taipei 宣布 Vera Rubin 进入全量产，确认 2026 H2 出货档期（详见 2026-06-01.md §3）。旗舰 NVL72 含 **72 颗 Rubin GPU、36 颗 Vera CPU**（每颗 88 核 Armv9.2），agent 吞吐较上代 Grace Blackwell **10×**；配套首次量产的 Spectrum-X 共封装光学（CPO），称较传统光模块省电 **5×**，面向百万 GPU 级 AI 工厂。黄仁勋把数据中心重述为"AI 工厂"经济：每 GW 资本开支由当前 $500–600 亿升向 **$800–1,000 亿**。下一关键节点：**今秋**首批出货，是 AI labs 与超大规模厂商 1 GW 级算力落地的时间锚。

（来源：[NVIDIA Newsroom](https://nvidianews.nvidia.com/news/vera-rubin-full-production-agentic-ai-factory) · [SiliconANGLE](https://siliconangle.com/2026/06/01/five-thoughts-nvidia-ceo-jensen-huangs-gtc-taipei-2026-keynote/) · [VideoCardz](https://videocardz.com/newz/nvidia-vera-rubin-nvl72-detailed-72-gpus-36-cpus-260-tb-s-scale-up-bandwidth)）

### 📌 NVIDIA RTX Spark（原 N1X）正式发布，主打 Windows 本地 agent

今秋随 Dell、Surface 等 6 家厂商上市，本地可跑 **120B** 参数模型。代号 N1X 的笔记本芯片 **6/1** GTC Taipei 以 **RTX Spark** 之名落地（详见 2026-06-01.md §1）。它把含 **6,144** CUDA 的 Blackwell RTX GPU 与 20 核 Grace CPU（MediaTek 共研）经 NVLink-C2C 整合，配 **128GB** 统一内存、**1 petaflop** AI 算力。卖点是 Windows 本地 agent：NVIDIA 与 Microsoft 联手推 OpenShell 运行时与新安全原语，让 OpenClaw 等在主力设备上安全运行；Adobe 为其重构 Photoshop、Premiere、AI 提速 **2×**。下一关键节点：**6/2–6/3** Microsoft Build 公布 Windows agent 开发者接口与安全原语细节。

（来源：[NVIDIA Newsroom](https://nvidianews.nvidia.com/news/nvidia-microsoft-windows-pcs-agents-rtx-spark) · [Tom's Hardware](https://www.tomshardware.com/laptops/nvidia-enters-the-windows-pc-market-with-rtx-spark)）

## 3. 机器人与物理 AI

### 📌 NVIDIA 开源 Isaac GR00T 人形参考设计，年底由 Unitree 发售

年底由 Unitree 发售，**75** 自由度，Ai2、Stanford 等首批采用。NVIDIA **6/1** 发布首个开源人形机器人参考设计 Isaac GR00T Reference Humanoid，延续 Jetson Thor 物理 AI 主线（详见 2026-06-01.md §2）。硬件以 **Unitree H2 Plus** 为身（约 1.8 米、150 磅、31 自由度）加双 Sharpa Wave 触觉五指手（22 自由度），合计 **75** 自由度；脑为 Jetson AGX Thor T5000（Blackwell、**2,070** FP4 TFLOPS）。同日 NVIDIA 还发布物理 AI 开放基础模型 **Cosmos 3** 与一批开源 agent 工具。下一关键节点：参考设计**年底**由 Unitree 上市，Unitree G1 工作流将很快上 GitHub 与 Hugging Face。

## 4. 自动驾驶

### 📌 NVIDIA DRIVE Hyperion 扩为全球 robotaxi 平台

Foxconn 高雄 **2028** 开跑、Uber 慕尼黑年内落地，L4 生态扩到亚欧中东四区。NVIDIA **6/1** 宣布 DRIVE Hyperion L4-ready 平台扩盟，基于 NVIDIA Halos 安全栈把车厂、Tier 1、出行平台并入同一 L4 框架。Foxconn 先在台湾**高雄**部署 robotaxi、**2028** 起跑（机场—市区起步、后接高铁走廊）；VinFast 偕 Autobrains 做东南亚 L4；Uber 偕 Autobrains 在**慕尼黑**年内启动 robotaxi；HUMAIN 把 robotaxi 带入沙特。同日另发面向 robotaxi 的开源推理模型 **Alpamayo 2**。下一关键节点：Uber 慕尼黑项目的合作车厂预计 **2026 下半年**公布，是该生态从试点转向商业网络的检验点。
