## 2. 芯片与半导体

### 🔥 NVIDIA Vera Rubin 进入全量产，今秋出首批货

全球 **150** 家台厂、**350+** 工厂、30 国已在量产爬坡，今秋出首批货。NVIDIA 在 **6/1** GTC Taipei keynote 宣布 Vera Rubin 平台进入全量产，确认此前预告的 2026 H2 出货档期（详见 2026-06-01.md §3）。Vera Rubin 是 POD 级架构——5 个机柜作为一台 AI 超算运行；旗舰 NVL72 含 **72 颗 Rubin GPU + 36 颗 Vera CPU**（每颗 88 核 Armv9.2），agent 吞吐较上代 Grace Blackwell **10×**。配套首次量产的 Spectrum-X Ethernet Photonics（共封装光学 CPO），称较传统光模块省电 **5×**、AI 正常运行时间 5×，面向百万 GPU 级 AI 工厂。黄仁勋把数据中心重述为"AI 工厂"经济：单站点迈向 1 GW、每 GW 资本开支从 **$500–600 亿** 升向 **$800–1,000 亿**，故每瓦 token 吞吐即营收。首批云厂含 Microsoft Azure、CoreWeave、Lambda、OCI。下一关键节点：**今秋**首批生产出货，是 AI labs 与超大规模厂商 1 GW 级算力落地的时间锚。

（来源：[NVIDIA Newsroom](https://nvidianews.nvidia.com/news/vera-rubin-full-production-agentic-ai-factory) · [SiliconANGLE](https://siliconangle.com/2026/06/01/five-thoughts-nvidia-ceo-jensen-huangs-gtc-taipei-2026-keynote/) · [VideoCardz](https://videocardz.com/newz/nvidia-vera-rubin-nvl72-detailed-72-gpus-36-cpus-260-tb-s-scale-up-bandwidth)）

### 📌 NVIDIA RTX Spark（原 N1X）正式发布，主打 Windows 本地 agent

今秋随 Dell、HP、Lenovo、Surface、MSI 上市，本地可跑 **120B** 参数模型。业界叫了三年的 N1X 在 **6/1** GTC Taipei 以 **RTX Spark** 之名落地（详见 2026-06-01.md §1 预热）。这颗超级芯片含 **6,144** CUDA 的 Blackwell RTX GPU + 20 核 Grace CPU（与 MediaTek 共研），经 NVLink-C2C 互连、**128GB** 统一内存、TSMC 3nm，AI 算力 **1 petaflop**，可本地运行 120B 参数、100 万 token 上下文的大模型。卖点是 Windows 本地 agent：NVIDIA 与 Microsoft 联手推 OpenShell 运行时 + 新 Windows 安全原语，让 Hermes Agent、OpenClaw 等在主力设备上安全运行；Adobe 把 Photoshop、Premiere 为其重构、称 AI 提速 **2×**。企业侧另有 DGX Station for Windows，把万亿参数级 AI 超算放上桌面。下一关键节点：**6/2–6/3** Microsoft Build 公布 Windows agent 的开发者接口与容器化安全原语细节。

（来源：[NVIDIA Newsroom](https://nvidianews.nvidia.com/news/nvidia-microsoft-windows-pcs-agents-rtx-spark) · [Tom's Hardware](https://www.tomshardware.com/laptops/nvidia-enters-the-windows-pc-market-with-rtx-spark)）

## 3. 机器人与物理 AI

### 📌 NVIDIA 开源 Isaac GR00T 人形参考设计，年底由 Unitree 发售

年底由 Unitree 发售，Ai2、ETH Zurich、Stanford、UC San Diego 已定为科研平台。NVIDIA **6/1** 发布首个开源人形机器人参考设计 Isaac GR00T Reference Humanoid，延续 Jetson Thor 物理 AI 主线（详见 2026-06-01.md §2）。硬件以 **Unitree H2 Plus** 为身（约 1.8 米、150 磅、31 自由度）+ 双 Sharpa Wave 触觉五指手（22 自由度），合计 **75** 自由度；脑为 Jetson AGX Thor T5000（Blackwell、**2,070** FP4 TFLOPS、128GB）。配套 Isaac GR00T 软件栈（Teleop 采数据、Isaac Sim / Lab 仿真训练、Isaac ROS 部署），研究者保留自有机器人数据与日志。同日 NVIDIA 还发布物理 AI 开放基础模型 **Cosmos 3** 与一批开源 agent 工具。下一关键节点：参考设计**年底**由 Unitree 上市，Unitree G1 的 GR00T 工作流将很快上 GitHub / Hugging Face。
