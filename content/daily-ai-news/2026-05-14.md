## 1. 主要 AI Labs 动态

### 🔥 Anthropic 启动 $300–500 亿新融资，估值 $900–950 亿

5/13 周三 Bloomberg、Sherwood、TechCrunch 等多家媒体援引消息源报道，Anthropic 正与潜在投资人接触一轮规模约 **$300–500 亿**、估值 **$900–950 亿** 的新融资。若敲定将首次超过 OpenAI 2026/3 轮 $852 亿估值，使 Anthropic 成为全球估值最高的私营公司。Anthropic 官方 newsroom 截至本日尚未发布融资公告，消息全部来自匿名信源。

Dario Amodei 在媒体口径中给出的收入数据是"**~$300 亿** 年化 ARR，今年内有望再增长约 80 倍"。需要注意的是 ARR 与"年化使用量增长"是两个口径，此前 5/6 大会上 Amodei 提到的"Q1 单季 80 倍"是指年化基础下的使用量+营收综合增速，与"年内再 80 倍"的前瞻指引不应直接相加。IPO 时间窗最早 2026/10。

（来源：[Sherwood News](https://sherwood.news/tech/anthropic-in-talks-for-funding-at-a-valuation-as-high-as-950-billion-which-would-make-it-bigger-than-openai/) · [TechCrunch (Ramp data)](https://techcrunch.com/2026/05/13/anthropic-now-has-more-business-customers-than-openai-according-to-ramp-data/) · [Bloomberg via Yahoo Finance](https://finance.yahoo.com/news/anthropic-talks-raise-30-billion-210804604.html)）

### 🔥 Ramp 数据：Anthropic 首次反超 OpenAI 企业付费客户份额

Ramp 5/13 发布的 5 月 AI Index 基于约 50,000 家美国公司的费用数据，显示 **Anthropic 的付费企业客户份额已达 34.4%**，超过 OpenAI 的 **32.3%**。2025/5 时这两个数字还是 9% 对 33%——12 个月里 Anthropic 涨了约 25 个百分点，OpenAI 微跌 1 个百分点。金融、科技、专业服务三个高采纳行业里 Anthropic 的领先更明显。

VentureBeat 在跟进文章中指出 Anthropic 的领先并不稳固，OpenAI 的 ChatGPT Business、Microsoft 渠道、以及 OpenAI 5/11 推出的 Deployment Company 都是潜在反扑路径。

## 2. AI Agent

### 📌 Cursor 5/13：多 repo 云端 Agent 环境 + Bugbot 三档 effort

Cursor 5/13 在 changelog 发布团队级 Agent 基础设施更新。**Multi-Repo Cloud Agents Environments** 允许在单一环境内组合多个仓库，跨 session 复用；环境配置走 Dockerfile-as-code 路径，支持 build secrets，缓存命中时构建快 70%，每个环境有版本历史和审计日志，admins 可限制 rollback 权限。

同时发布的 **Bugbot Effort Levels**（5/11 推出）让 Teams admin 与个人付费用户在 Default / High / Custom 三档之间选择 PR review 的推理深度。Cursor 官方给出的数据：Default 模式平均每次 run 找到 0.7 个 bug，High 模式 0.95 个；79% 以上的 bug 在合并时会被开发者真正修复。

（来源：[Cursor Changelog: Dev Environments](https://cursor.com/changelog/05-13-26) · [Cursor Changelog: Bugbot Effort](https://cursor.com/changelog/05-11-26)）

## 5. 自动驾驶

### 🔥 Waymo 5/13：服务区单次扩展 200–300 平方英里，总覆盖突破 1,400 平方英里

Waymo 5/13 周三宣布在现有 11 个运营城市内新增 **200–300 平方英里** 覆盖面积，单次扩展面积大于 Rhode Island 全州，使累计服务区超过 **1,400 平方英里**。扩张顺序从 Miami 开始，Bay Area、Houston、Austin、Atlanta 紧随。Waymo 当前周付费乘次约 50 万次，累计约 2,000 万次，车队约 3,000 辆，目标 2026 年内做到每周 100 万乘次，伦敦是计划中的海外首站。
