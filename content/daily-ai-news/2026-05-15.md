## 1. 主要 AI Labs 动态

### 🔥 Claude Platform on AWS 完成 17 区域 GA，原生集成 IAM 与 CloudTrail

5/14 周四 AWS What's-New 与 Anthropic 联合发文确认 **Claude Platform on AWS** 完成 GA。AWS 客户可通过现有账户直接接入 Anthropic 原生 Claude Platform，包括 APIs、Console 与早期 Beta 功能，无需独立账号、计费、配额跟踪。覆盖能力含 Messages API、Managed Agents、Advisor Tool、Web Search、MCP Connector、Agent Skills、Code Execution、Files API。账号体系融入 AWS：认证走 IAM、审计走 CloudTrail、计费走标准 AWS invoice。区域覆盖北美、南美、欧洲、亚太共 **17 个**，与既有 AWS Bedrock 上 Claude 模型路径并行。

（来源：[AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/05/claude-platform-aws/) · [Anthropic Blog](https://claude.com/blog/claude-platform-on-aws) · [The New Stack](https://thenewstack.io/anthropics-claude-platform-comes-to-aws/)）

*披露：本报告由 Claude 撰写，Anthropic 为 Claude 开发方。*

### Claude Code v2.1.142：Fast Mode 默认升级至 Opus 4.7

5/14 周四晚 22:55 UTC Anthropic 发布 Claude Code **v2.1.142**。Fast Mode 默认模型由 Opus 4.6 升级至 **Opus 4.7**，可通过环境变量 `CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE=1` 回锁旧版。`claude agents` 子命令新增 8 类 flag 配置后台分发会话，含 `--add-dir`、`--settings`、`--mcp-config`、`--model`、`--permission-mode`、`--effort` 等。Hook JSON 输出新增 `terminalSequence` 字段，允许 hook 在无控制终端环境下触发桌面通知或系统铃声。

（来源：[Claude Code GitHub Releases](https://github.com/anthropics/claude-code/releases) · [Claude Code What's New](https://code.claude.com/docs/en/whats-new)）

*披露：本报告由 Claude 撰写，Anthropic 为 Claude 开发方。*

## 2. AI 产品更新

### 📌 Codex Mobile 5/14 周活破 400 万，三端齐发承接远程开发会话

5/14 周四 OpenAI 在 ChatGPT 移动 app 内开放 **Codex Mobile** Preview，iOS、iPad、Android 三端齐发，免费与 Go 等所有套餐用户均可用。OpenAI 同日披露 Codex 周活用户已突破 **400 万**。

核心交互：ChatGPT 移动 app 充当远程开发环境桥接终端，可连接 macOS 上的 Codex 实例（Windows 即将支持），手机端可跨线程查看、审批命令、切换模型、回看截图、终端输出、diff 与测试结果，所有文件与 credentials 仍留在执行机器上。前日 2026-05-14.md 已首发报道，本条聚焦周活数据。

（来源：[TechCrunch](https://techcrunch.com/2026/05/14/openai-says-codex-is-coming-to-your-phone/) · [OpenAI Blog](https://openai.com/index/work-with-codex-from-anywhere/) · [Engadget](https://www.engadget.com/2173235/openai-brings-its-codex-coding-app-to-mobile/)）

## 4. 芯片与半导体

### 🔥 TSMC 把 2030 全球芯片市场指引上修至 $1.5 万亿，AI 与 HPC 占 55%

5/14 周四 TSMC 在台北技术论坛把 2030 年全球半导体产值指引上修至 NT$47.27 万亿（约 **$1.5 万亿**）。结构上 AI 与高性能计算将占 **55%**，超过智能手机 20%、汽车 10%、IoT 10%。TSMC 同时给出自身 2nm 与 A16 工艺在 2026–2028 年 **70% CAGR** 的产能爬坡预期。TSMC 把 AI 而非智能手机定位为半导体长期主增长引擎是十年来首次，意味着 NVIDIA、AMD、Marvell、Broadcom 等 AI 客户的台积电产能在中长期将持续被定价为超额稀缺资源，与 5/13 Bank of America 上调 NVIDIA 目标价至 $320 的算力短缺论形成同向支撑。

（来源：[Taiwan News](https://www.taiwannews.com.tw/news/6362348) · [Republic World](https://www.republicworld.com/tech/ai-to-overtake-smartphones-as-semiconductor-growth-engine-global-chip-output-to-hit-15-trillion-by-2030-2026-05-14-124237) · [Benzinga](https://www.benzinga.com/markets/equities/26/05/52551643/taiwan-semiconductor-sees-global-chip-market-hitting-1-5-trillion-by-2030-amid-ai-boom)）
