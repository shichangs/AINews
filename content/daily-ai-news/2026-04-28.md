## 1. 主要 AI Labs 动态

### 🔥 Microsoft × OpenAI 合作协议重写：Microsoft 不再付收入分成 + 非独家 IP + AGI 条款移除
**Microsoft + OpenAI** 于 **4 月 27 日**同步发布**合作"下一阶段"**官方公告——这是 **2025 年 10 月 OpenAI PBC 重组**之后**双方协议的最大幅度修订**，且由 **CNBC、Bloomberg、Microsoft 官方博客、OpenAI 官方博客**同时披露：
- **收入分成**：**Microsoft 不再向 OpenAI 支付任何收入分成**；**OpenAI 向 Microsoft 的分成保留至 2030 年**——**比例不变 + 设总额上限**（**total cap**），**与 OpenAI 技术路径无关**——**Microsoft 失去自有 AI 产品**（**Copilot / Azure OpenAI**）**对 OpenAI 的"反向贡献"**
- **IP 许可非独家化**：Microsoft 对 **OpenAI 模型 + 产品 IP** 的许可**延长至 2032 年**，但**首次改为非独家**——**Anthropic Claude / Amazon Bedrock / Google Cloud 等**理论上**均可获得 OpenAI IP 同等条件**
- **AGI 条款移除**：**Microsoft 不再需要在 OpenAI 宣告达到 AGI 时做"特殊响应"决定**——**双方"AGI 触发条款"**（自 2019 年合作协议遗留）正式作废
- **多云出货**：**OpenAI 可跨任意云提供商出货产品** ——**与 Amazon AWS 4 月 16 日 $500 亿 / 7 年合作 + Google Cloud / Oracle 多云布局**官方对齐
- **Azure 优先**：**Microsoft 仍是 OpenAI 主要云伙伴 + OpenAI 产品默认 Azure 首发**——**除非 Microsoft 不能或不愿支持所需能力**

**市场反应**：**Microsoft 股价**周一**短线下挫**——**失去 OpenAI 反向收入分成 + 失去独家 IP**双重打击；**Anthropic、Cohere、xAI**等"非 Microsoft 阵营"被解读为**潜在赢家**；**Salesforce、Slack、Notion**等**大量绑定 OpenAI 的 SaaS 厂商**视该协议为"**多云供给保障**"。

**联动信号**：与 **Anthropic 4 月 7 日 ARR $300 亿超越 OpenAI $250 亿（Anthropic 首次反超）+ Anthropic × Google + Broadcom 算力扩张 + Google Cloud Next '26 Workspace Intelligence + Cohere × Aleph Alpha 4/24 合并 $200 亿估值**叠加——**Microsoft × OpenAI"独占式同盟"正式终结**，**前沿 LLM 进入"多云 + 多伴 + 非独家"主导格局**。**Microsoft Q3 财报（5 月初）**前**资本市场对 Microsoft AI 战略再定价**已开始。（来源：[OpenAI 官方公告](https://openai.com/index/next-phase-of-microsoft-partnership/) · [Microsoft 官方博客](https://blogs.microsoft.com/blog/2026/04/27/the-next-phase-of-the-microsoft-openai-partnership/) · [CNBC](https://www.cnbc.com/2026/04/27/openai-microsoft-partnership-revenue-cap.html) · [Bloomberg](https://www.bloomberg.com/news/articles/2026-04-27/microsoft-to-stop-sharing-revenue-with-main-ai-partner-openai) · [TechCrunch](https://techcrunch.com/2026/04/27/openai-ends-microsoft-legal-peril-over-its-50b-amazon-deal/) · [Yahoo Finance](https://finance.yahoo.com/sectors/technology/article/microsoft-openai-rewrite-partnership-to-eliminate-exclusive-model-access-change-revenue-sharing-140522820.html) · [Stocktwits Microsoft Loses Exclusive](https://stocktwits.com/news-articles/markets/equity/microsoft-loses-exclusive-license-openai-models-taps-anthropic-claude/cZBNAqiRePg)）
