## 2. AI Agent

### 🔥 Claude Code 上线 Routines：本机离线时也能跑的可重复 Agent 任务
**Anthropic** 于 4 月 14 日为 **Claude Code** 推出 **Routines**（研究预览版）。核心卖点是 Routines 运行在 Claude Code 的**云端 Web 基础设施**上，而非用户本机——这意味着即使开发者关闭了笔记本电脑或离线，**已调度的可重复 Agent 任务（如每日 lint/测试、周报生成、跨仓库迁移）仍能自动执行**并在用户重新在线后同步结果。同时本次更新还带来：`EnterWorktree` 工具新增路径参数、`PreCompact` 钩子可阻断压缩、后台插件监控器、以及 Claude Desktop 的大幅改版（支持**多 Claude 会话并排**、集成终端、文件编辑、HTML/PDF 预览）。该更新将 Claude Code 从"交互式 Copilot"进一步推向"持续在岗的 Agent"。（来源：[9to5Mac](https://9to5mac.com/2026/04/14/anthropic-adds-repeatable-routines-feature-to-claude-code-heres-how-it-works/) · [Claude Code Changelog](https://code.claude.com/docs/en/changelog)）

### NVIDIA 在 GTC 2026 推出企业级 AI Agent 平台，首批 17 家合作方
**NVIDIA** 在 GTC 2026 上公开企业级 **AI Agent 平台**，首批公布 **Adobe、Salesforce、SAP** 等 **17 家**合作方作为早期采用者。平台主体包括 **Agent Toolkit**（含 Nemotron 模型、AI-Q 蓝图、OpenShell 运行时、cuOpt 优化库）。NVIDIA 同时披露 Klarna、Uber、LinkedIn 等企业在 LangGraph 上的调用已超 **4,700 万次**，Agent 工作流在企业侧已进入规模化阶段。（来源：[VentureBeat](https://venturebeat.com/technology/nvidia-launches-enterprise-ai-agent-platform-with-adobe-salesforce-sap-among-17-adopters-at-gtc-2026/)）
