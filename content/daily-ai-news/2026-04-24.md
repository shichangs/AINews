## 2. AI 产品更新

### 🔥 Cursor 3.2 上线：/multitask 异步子 Agent + Worktrees + Multi-root Workspaces
**Cursor** 于 4 月 24 日正式发布 **Cursor 3.2**——距 3 月 Cursor 3 主版本约六周，是其异步 Agent 体验的第二轮重构。官方 changelog（"Multitask, Worktrees, and Multi-root Workspaces"）涵盖三项主能力：

- `/multitask`：单工作区并行运行多个异步子 Agent，长任务 + 短任务穿插不再串行排队；可对已在队列中的消息直接转为 multitask 触发；自动把大任务拆分为小子任务交由 fleet 子 Agent 共跑；
- 改良 Worktrees：原生集成 Git worktree，多分支并行编辑在 Agents Window 中后台运行隔离任务，准备好后一键把分支 promote 到前台测试；
- Multi-root Workspaces：单 Agent 会话可定位多文件夹 reusable workspace，跨仓库（前端 / 后端 / 共享库）编辑无需切换会话或重新指定 Agent。

3.2 把 Cursor 从"逐次同步交互的 IDE Agent"明确推进至**异步并行的 Agent 执行平台**，方向与 OpenAI Codex 云端 Agent / Anthropic Claude Code subagents / GitHub Copilot Workspaces 一致——但多任务编排前置到 IDE 端仍是差异化策略。结合本周 SpaceX 与 Cursor 签 $600 亿可转债 + 收购选择权、xAI × Mistral × Cursor 三方合作传闻，Cursor 正在资本侧加速绑定 Musk 系。（来源：[Cursor Changelog](https://cursor.com/changelog/04-24-26) · [Cursor Community Forum](https://forum.cursor.com/t/multitask-worktrees-and-multi-root-workspaces/158954)）

### 🔥 DeepSeek V4-Pro / V4-Flash 开源：1.6T MoE + 1M 上下文 + Hybrid Attention
**DeepSeek** 于 4 月 24 日通过官方 X 与 API Docs 公告正式发布 V4-Pro / V4-Flash 预览版，模型权重在 Hugging Face 同步开源（修改 MIT）：
- **DeepSeek-V4-Pro**：1.6T 总参数 / **49B 激活**的 MoE；
- **DeepSeek-V4-Flash**：284B 总参数 / **13B 激活**的 MoE；
- 两款均原生 **1M token 上下文**——采用 Hybrid Attention（Compressed Sparse Attention + Heavily Compressed Attention 双层组合），在 1M 上下文下单 token 推理 FLOPs 仅为 V3.2 的 **27%**、KV cache 仅 **10%**——大幅缓解长上下文部署的算力与显存瓶颈；
- DeepSeek API 定价：**$1.74 / $3.48** 每百万输入 / 输出 tokens（缓存命中 $0.145）——约为 Claude Opus 4.7（$5/$25）输出价的 **1/7**、GPT-5.5（$5/$30）输出价的 **1/9**；
- 旧标识符 `deepseek-chat` 与 `deepseek-reasoner` 将于 **2026 年 7 月 24 日**下线。

V4-Pro 在 agentic、编码、数学等公开评测上自报达到与 GPT-5.5 / Claude Opus 4.7 同档"对位"水平；社区独立复测仍在进行中。结合 4 月 23 日腾讯 Hy3 Preview 开源（295B MoE / 21B 激活 / 256K 上下文），中国头部模型 2026 Q2 集中进入"第二代长上下文 + 高性价比 MoE"交付期，与 Gemini 3.1 Pro / Claude Opus 4.7 的能力时差被压窄至 3–6 个月。（来源：[DeepSeek API Docs](https://api-docs.deepseek.com/news/news260424) · [CNBC](https://www.cnbc.com/2026/04/24/deepseek-v4-llm-preview-open-source-ai-competition-china.html) · [Hugging Face](https://huggingface.co/deepseek-ai/DeepSeek-V4-Pro)）

## 4. AI Agent

### Hugging Face ml-intern 进入开发者主流视野：开源 ML 工程 Agent
**Hugging Face** 在本周（4 月 21 日首发，4 月 24 日前后被 MarkTechPost、Medium、ETIH 等社区媒体广泛报道与采纳）正式开源 **ml-intern**——端到端自动化 LLM 后训练的开源 AI Agent，基于自家 smolagents 框架。覆盖能力包括：文献检索、数据集发现、训练脚本生成与执行、迭代评测四大子任务；并可作为单 Agent 自闭环或与企业 MLOps 集成。官方演示：把 Qwen3-1.7B 基础模型在 GPQA 上从 baseline 约 10% 提升到 **32%**，总耗时不足 **10 小时**，3 小时即跨过 27.5%。Hugging Face 自报：在科学推理基准上击败 Claude Code、在医疗基准上比 Codex 高约 **60%**——独立复测仍待社区跟进。结合 OpenAI Workspace Agents、Google Workspace Intelligence、Anthropic Computer Use 同周节奏，"自训练 Agent / 自动 ML 工程师"进入 2026 Q2 主流议题。（来源：[MarkTechPost](https://www.marktechpost.com/2026/04/21/hugging-face-releases-ml-intern-an-open-source-ai-agent-that-automates-the-llm-post-training-workflow/) · [Hugging Face GitHub](https://github.com/huggingface/ml-intern) · [EdTech Innovation Hub](https://www.edtechinnovationhub.com/news/hugging-face-releases-ml-intern-the-ai-agent-teaching-itself-to-beat-claude-code-on-scientific-reasoning)）
