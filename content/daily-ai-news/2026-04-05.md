## 2. AI Agent

### 📌 MCP 安装量突破 9,700 万，成为 agentic AI 事实标准

Anthropic 主导的 **Model Context Protocol (MCP)** 近日公布阶段性数据：截至 **3 月 25 日**合计安装量已突破 **9,700 万次**（Python 与 TypeScript SDK 合计，较 2024 年底上线时的约 10 万次近千倍增长）：

- **生态覆盖**：**OpenAI、Google、xAI、Mistral、Cohere** 均已在其 API 中加入 MCP 支持，公共和企业部署中**活跃 MCP server 数量超过 1 万个**；
- **企业落地**：Q1 2026 被视为**Fortune 500 企业将 agentic AI 从试点推向生产部署**的关键季度；
- **治理变化**：MCP 已被捐赠给 Linux 基金会下属的 **Agentic AI Foundation**，和 **OpenAI AGENTS.md**、**Block goose** 等一同构成开放治理基线。

## 3. AI 产品更新

### 🔥 Google Gemma 4 正式发布：四种规格全面开源，Apache 2.0 授权，31B 登 Arena 榜第三

4 月 2 日，**Google DeepMind** 发布 **Gemma 4** 开源模型家族，迄今最强开放模型，专为推理与 Agent 工作流优化：

- **四种规格**：
  - **E2B / E4B**（Effective）：为智能手机、平板等边缘设备设计，推理时激活参数仅 2B/4B，大幅节省内存与电池；
  - **26B MoE**（Mixture of Experts）：Arena AI 文字榜**第六**；
  - **31B Dense**：Arena AI 文字榜**第三**，比同尺寸模型能力高出数倍；
- **能力升级**：原生支持**视觉与音频**输入、高达 **256K** 上下文窗口、超过 **140 种语言**，擅长复杂逻辑、代码生成和多步 Agent 任务；
- **授权突破**：改为 **Apache 2.0** 许可证（此前版本为限制性许可），企业可放心商业部署、微调、集成，无需担忧 Google 单方面撤回访问权；
- **多平台可用**：Google AI Studio、AI Edge Gallery、Hugging Face、Kaggle、Ollama 均已支持；Android AICore 开发者预览版同步上线。

（来源：[Google Blog](https://blog.google/innovation-and-ai/technology/developers-tools/gemma-4/) · [The Register](https://www.theregister.com/2026/04/02/googles_gemma_4_open_weights/) · [Xinhua](https://english.news.cn/northamerica/20260403/3c3bdb9b42054ebd887a5daf5a8cfa2c/c.html)）

### 🔥 Microsoft 发布三款自研 MAI 模型：转录、语音、图像全覆盖

4 月 2 日，**Microsoft** 正式宣布三款自研多模态基础模型，通过 Microsoft Foundry 和 MAI Playground 提供服务，被外界视为微软深化**摆脱对 OpenAI 依赖**的重要一步：

- **MAI-Transcribe-1**（语音转文本）：在 **FLEURS** 基准测试 25 种语言平均词错率（WER）仅 **3.8%**，批量转录速度是此前 Azure Fast 方案的 **2.5 倍**；起步价 **$0.36/小时**；
- **MAI-Voice-1**（文字转语音）：旗舰级语音生成模型，可 1 秒内生成 **60 秒**高质量音频，情感表达丰富、保留说话人音色；起步价 **$22/百万字符**；
- **MAI-Image-2**（图像生成）：登 Arena.ai 图像榜前三，已驱动 Copilot 图像生成大幅提速；起步价 **$5/百万 token**（文本输入），**$33/百万 token**（图像输出）；
- **战略意义**：自研模型逐步覆盖语音、图像、转录等多模态场景，使微软能在不依赖 OpenAI API 的情况下为 Azure 和 Copilot 产品线提供核心能力。

（来源：[Microsoft AI](https://microsoft.ai/news/today-were-announcing-3-new-world-class-mai-models-available-in-foundry/) · [VentureBeat](https://venturebeat.com/technology/microsoft-launches-3-new-ai-models-in-direct-shot-at-openai-and-google) · [TechCrunch](https://techcrunch.com/2026/04/02/microsoft-takes-on-ai-rivals-with-three-new-foundational-models/)）

### 🔥 Cursor 3.0 正式上线：并行多 Agent、Design Mode，编程 IDE 迎来最大架构升级

4 月 2 日，**Cursor** 发布 **3.0 版本**，这是自编辑器上线以来最大的一次架构重塑：

- **Agents Window（核心新功能）**：全新界面以 Agent 编排为中心，支持**并行运行多个 Agent**，可跨代码仓库、本地环境、Git worktree、云端和远程 SSH 同步工作；
- **Design Mode**：在 Agents Window 中直接对浏览器中的 UI 元素进行**标注与指向**，让 Agent 精准理解视觉反馈，加速前端迭代；
- **Agent Tabs**：支持多个聊天并排或网格布局查看，方便对比不同 Agent 的工作结果；
- **新命令**：`/worktree` 自动创建隔离 Git worktree，`/best-of-n` 支持多模型并行对比；
- **Composer 2 模型**：前沿级编程模型，定价 **$0.50/M input** · **$2.50/M output**；
- **Real-Time RL**：基于真实用户交互的实时强化学习，最快**每 5 小时**更新一次 Composer 模型 checkpoint；
- **向下兼容**：所有现有订阅用户**免费升级**。

## 7. 医疗 AI

### FDA 重新定义"突破性 AI 医疗器械"：偏向"医生做不到"的多任务模型

STAT News 4 月 2 日报道，美国 **FDA** 对 AI 医疗器械的 **Breakthrough Designation** 认定标准出现**明显漂移**：

- **新门槛**：单纯"辅助医生提升诊断效率"的算法已**不再足以**获得突破性认定，FDA 倾向于批准能够**同时解决多种问题**的 AI，例如**从单张影像检出多种癌症**，或**预测癌症/心衰的死亡风险**；
- **政策协同**：2026 年联邦 AI 行动计划三大支柱之一是**为 AI 医疗器械提供更好的医保报销激励**，新版 Medicare 医师收费标准明确给予使用合规 AI 工具的服务更高补偿；
- **现状**：FDA 累计批准 AI/ML 医疗器械**已逾 1,000 款**，其中影像分析占比开始下降。

## 10. 政策与监管

### 美国联邦法院：特朗普政府禁用 Anthropic AI 模型违反言论自由

一项美国联邦法院裁定指出，特朗普政府此前**禁止联邦政府系统使用 Anthropic AI 模型**的做法**违反了第一修正案言论自由保护**，相关禁令被撤销：

- **案情背景**：行政命令以"模型存在政治倾向"为由，要求政府部门不得采购或部署 Anthropic 模型；
- **裁决意义**：这是 2026 年以来 AI 领域**首个针对政府"选择性封禁模型"的言论自由裁决**，为其他可能被政治性下架的模型提供先例；

（来源：[Crescendo AI](https://www.crescendo.ai/news/latest-ai-news-and-updates)）
