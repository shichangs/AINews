📌 **Anthropic 赢得初步禁令：法官裁定五角大楼行为构成"第一修正案报复"**

3 月 26 日（昨日），联邦法官 **Rita Lin** 正式签发初步禁令，暂停五角大楼对 Anthropic 的"供应链风险"认定及特朗普总统禁止联邦机构使用 Claude 的行政指令：

- **核心裁决**：Lin 法官在书面裁定中写道："这看起来是经典的**第一修正案报复**"，并称"没有任何法律支持这种**奥威尔式**的做法——仅因一家美国公司对政府表达异议，就将其打上潜在敌对者和破坏者的标签"；
- **实际效果**：在诉讼期间，Anthropic 可继续与联邦政府机构和国防承包商开展业务合作；
- **背景回顾**：五角大楼要求 Anthropic 授予其对 Claude 模型的"不受限制的全用途访问"，Anthropic 拒绝后被贴上供应链风险标签；Anthropic 坚持要求其技术不得用于完全自主武器或国内大规模监控。

（来源：[CNBC](https://www.cnbc.com/2026/03/26/anthropic-pentagon-dod-claude-court-ruling.html) · [NPR](https://www.npr.org/2026/03/26/nx-s1-5762971/judge-temporarily-blocks-anthropic-ban) · [CNN](https://us.cnn.com/2026/03/26/business/anthropic-pentagon-injunction-supply-chain-risk) · [Axios](https://www.axios.com/2026/03/26/judge-temporarily-blocks-pentagon-ban-anthropic)）

## 2. AI Agent

**Cursor 推出自托管云 Agent：代码和执行环境完全留在企业内部网络**

3 月 25 日，**Cursor** 发布**自托管云 Agent（Self-Hosted Cloud Agents）**功能，允许企业在自有基础设施中运行 AI Agent，同时获得与 Cursor 托管云 Agent 相同的开发体验：

- **核心价值**：代码库、构建产物和密钥全部保留在企业网络内部，Agent 在本地执行工具调用，模型推理通过 API 完成；
- **功能对等**：支持隔离虚拟机、完整开发环境、多模型编排、插件等全部云 Agent 能力；
- **客户案例**：**Brex、Money Forward、Notion** 等已在使用；
- **目标场景**：满足金融、医疗等高度监管行业对数据不出境的合规要求。

（来源：[Cursor Blog](https://cursor.com/blog/self-hosted-cloud-agents) · [Cursor Changelog](https://cursor.com/changelog/03-25-26)）

**OpenAI 推出 Safety Bug Bounty 计划，聚焦 Agent 安全漏洞**

3 月 25-26 日，**OpenAI** 正式上线 **Safety Bug Bounty** 计划，作为原有 Security Bug Bounty 的补充，专门征集 AI 安全与滥用风险：

- **覆盖范围**：第三方 prompt 注入与数据窃取、Agent 劫持（Browser、ChatGPT Agent 等）、平台完整性绕过等；
- **排除范围**：不造成实际安全影响的内容策略绕过（如让模型说脏话）不在范围内；
- **托管平台**：通过 **Bugcrowd** 运营，研究人员可直接申请参与。

（来源：[OpenAI Blog](https://openai.com/index/safety-bug-bounty/) · [Infosecurity Magazine](https://www.infosecurity-magazine.com/news/openai-bug-bounty-ai-abuse-safety/)）

## 10. 政策与监管

**🔥 Meta 遭遇双重历史性判决：新墨西哥州 $3.75 亿 + 洛杉矶 $600 万**

3 月 24-26 日，**Meta** 在两起儿童安全诉讼中接连败诉，创下科技行业先例：

- **新墨西哥州案**（3 月 24 日）：陪审团认定 Meta 违反该州消费者保护法，其平台助长了**儿童性剥削**，判处 **$3.75 亿**民事罚款；这是美国**首个州政府**在此类案件中胜诉；
- **洛杉矶案**（3 月 26 日）：陪审团认定 Meta 和 YouTube 对**社交媒体成瘾**造成的伤害负有过失责任，判赔 **$600 万**（Meta 承担 $420 万，YouTube $180 万）；这是科技公司**首次被陪审团认定**需为成瘾性产品设计承担责任；
- **后续影响**：洛杉矶案仅是包含 **1,600+** 原告的合并诉讼中的第一起审判；Meta 表示将对两项判决提起上诉。

（来源：[CNBC](https://www.cnbc.com/2026/03/24/jury-reaches-verdict-in-meta-child-safety-trial-in-new-mexico.html) · [ABC News](https://abcnews.com/GMA/News/jury-returns-verdict-meta-youtube-landmark-social-media/story?id=131121474) · [NPR](https://www.npr.org/2026/03/24/g-s1-115019/new-mexico-meta-children-mental-health) · [Al Jazeera](https://www.aljazeera.com/news/2026/3/26/jury-finds-meta-youtube-liable-for-social-media-addiction-what-we-know)）
