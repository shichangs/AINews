## 3. AI Agent 与开发者生态

### Google 关停 Gemini CLI，10.5 万星开源工具改推闭源 Antigravity CLI

gemini 命令 6/18 起返回 HTTP 410，免费额度骤降 98%、CI 自动化中断。调用 **gemini** 的所有免费、AI Pro 与 Ultra 账户 6/18 起返回 **HTTP 410 Gone**，GitHub Actions、Makefile、cron 等自动化随之中断——距 Google 在 I/O 宣布该决定恰好 **30 天**。接替者 **Antigravity CLI**（命令 agy）是闭源 Go 二进制，主打异步多 agent 编排，但免费额度从约 1,000 次/日砍至约 **20 次/日**（降 98%）、且改为按周重置。迁移最大暗坑是 MCP 配置从 settings.json 内联改为独立 mcp_config.json、字段 url 改名 serverUrl，CI 会静默失败。争议在于：Gemini CLI 一年间累计 **10.5 万** GitHub star、合并约 6,000 个外部 PR，而贡献者签署的 CLA 让 Google 可将其代码并入闭源继任品。下一关键节点：Gemini Code Assist for GitHub 新装 6/18 起停止、全量服务约 7 月中旬终止，在 CI 调用 gemini 的团队须先手动迁移 mcp_config.json。

（来源：[Google Developers Blog](https://developers.googleblog.com/an-important-update-transitioning-gemini-cli-to-antigravity-cli/) · [TechTimes](https://www.techtimes.com/articles/318660/20260618/gemini-cli-shutdown-takes-effect-ci-cd-pipelines-break-go-based-antigravity-cli-arrives.htm)）

## 4. AI 产品

### Cursor 升级 Automations，云 agent 默认获 computer use

Cursor 自动化新增 5 个 GitHub 触发器，云 agent 默认获 computer use。**Cursor** 6/18 升级 Automations（常驻 agent 自动化），新增 **/automate** 技能：用自然语言描述任务，Cursor 自动配置触发器、指令与工具。触发方式扩展到 Slack 表情符号与 **5 个**新增 GitHub 事件（issue 评论、PR 行内评论、PR 评审提交、评审线程更新、Actions 运行完成）；由自动化启动的云 agent 现默认获得 computer use 工具，可自行操作电脑产出演示或交付物。另有三项改进：自动化可存为未完成态、默认可直接开 PR、可在 UI 删除记忆文件。下一关键节点：功能即时对全部用户生效，须更新到最新版 Cursor 启用。

## 5. 芯片与创业融资

### Architect Labs 携 $2,400 万出隐身，剑指 Broadcom 与 Marvell

Architect Labs 用 AI 端到端设计验证芯片，直取 Broadcom、Marvell 定制芯片生意。这家帕洛阿尔托初创 6/18 携 **$2,400 万**种子轮走出隐身，要复刻当年 TSMC 把"制造"对所有人开放的路径、缔造一个"无设计"（designless）的半导体业。**Kindred Ventures** 领投（创始人 Steve Jang 入董事会），TQ Ventures、Race Capital、Together Fund 跟投；据 Reuters，天使包括 Google DeepMind 首席科学家 **Jeff Dean** 及 OpenAI、Nvidia 高管。两位创始人 Ebrahim Hussain（前 Apple、Tesla 定制芯片）与 Aaditya Subedi（哈佛 AI 代码验证）在斯坦福相遇后退学创业；团队约 **18 人**、累计流片 **80+** 颗量产芯片，含 Intel、Meta、Anthropic、DeepMind、xAI 背景成员。下一关键节点：公司称已与半导体伙伴部署、预计年内在先进制程流片首批 AI 生成设计（尚无第三方验证）。
