## 4. AI Agent 与开发者生态

### Google 关停 Gemini CLI，10.5 万星开源工具改推闭源 Antigravity CLI

gemini 命令 6/18 起返回 HTTP 410，免费额度骤降 98%、CI 自动化中断。免费、AI Pro 与 Ultra 账户均受影响，GitHub Actions、cron 等随之停摆——距 I/O 宣布恰好 **30 天**。接替的 **Antigravity CLI**（agy）是闭源 Go 二进制、主打异步多 agent 编排，免费额度从约 1,000 次/日砍至约 **20 次/日**、改按周重置。迁移暗坑：MCP 配置须移入独立 mcp_config.json、url 字段改名 serverUrl，否则 CI 静默失败。**Gemini CLI** 一年累计 **10.5 万** GitHub star。下一关键节点：Gemini Code Assist for GitHub 全量服务约 **7 月中旬**终止。

（来源：[Google Developers Blog](https://developers.googleblog.com/an-important-update-transitioning-gemini-cli-to-antigravity-cli/) · [TechTimes](https://www.techtimes.com/articles/318660/20260618/gemini-cli-shutdown-takes-effect-ci-cd-pipelines-break-go-based-antigravity-cli-arrives.htm)）
