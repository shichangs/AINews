## 2. 主要 AI Labs 动态

### 🔥 OpenAI 5/7 发布 GPT-5.5-Cyber，限量预览给已审核网络防御者

OpenAI 在 5/7 周四上线 **GPT-5.5-Cyber**，作为 GPT-5.5 的网络安全变体，通过 Trusted Access for Cyber 项目以有限预览形式开放给已审核的关键基础设施防御团队。OpenAI 在官方页面说明该变体在 GPT-5.5 之上"对安全相关任务更宽松"，主要场景包括漏洞识别与分诊、补丁验证、恶意软件分析、攻击逆向工程；最高一档 Trusted Access 通过者可使用守备约束相对放松的版本，用于 bug hunting / malware 研究等防御性工作。

公司同步称首批用户来自电网、医院、银行等关键基础设施运营方，强调"目标不是显著提升攻击能力，而是让既有防御工作流跑得更顺"。该动作与 5/5 CAISI 三家加入预审、5/4 白宫 EO 提案合并构成本月美方"上市前自愿审 + 防御侧定向开放"双轴。Axios 报道指这是 OpenAI 首次为单一垂直行业提供独立模型（来源：[OpenAI Trusted Access for Cyber 公告](https://openai.com/index/gpt-5-5-with-trusted-access-for-cyber/) · [CNBC 5/7](https://www.cnbc.com/2026/05/07/openai-rolls-out-new-gpt-5point5-cyber-to-vetted-cybersecurity-teams.html) · [Axios 5/7](https://www.axios.com/2026/05/07/openai-gpt-55-cybersecurity-model)）。

## 4. AI 产品更新

### 🔥 OpenAI 5/7 ChatGPT Ads 自助 Ads Manager 美区 beta，新增 CPC 出价，预告 5 国扩盘

OpenAI 5/7 周四公布 ChatGPT 广告业务的多项推进：面向美国广告主的自助 **Ads Manager** 进入 beta；同步上线 **CPC**（每次点击成本）出价，让广告支出更直接对齐点击后行动；并预告未来数周把 ChatGPT 广告 pilot 扩至英国、日本、韩国、巴西、墨西哥 5 国。整套披露来自新任 OpenAI VP of Monetization Benji Shomair 5/7 的 LinkedIn 帖。

### 📌 Microsoft 5/7 Windows 11 Copilot 战略收缩：从 Snipping Tool / Photos / Notepad / Widgets 移除入口

Microsoft 在 5/7 周四通过 Windows Latest 报道与新任 Copilot EVP **Jacob Andreou**（前 Snap 高管）X 帖（随后删除）的形式，确认 Windows 11 中 Copilot 入口的战略性收缩：Snipping Tool、Photos 已完全移除 "Ask Copilot" 按钮；Notepad 中的 Copilot 标识改名为更低调的 "Writing tools"（底层仍由 AI 驱动）；Widgets 同样进入清单。Andreou 表态原文："关键是把 Copilot 从未达到承诺的地方移除。"

该收缩沿用 4 月 9 日起逐步推进的"减少不必要 Copilot 入口"路线，并与 5/5 Xbox Copilot 全砍、5/6 Dragon Copilot for Nurses 落地同步 —— Microsoft 5 月 Copilot 战略呈现"B 端扩张 + 消费应用层收缩"双向收口（来源：[Windows Latest 5/7](https://www.windowslatest.com/2026/05/07/windows-11-pulls-back-ai-as-microsoft-plans-to-remove-copilot-where-it-doesnt-meet-its-promise/) · [PC Gamer 报道](https://www.pcgamer.com/software/operating-systems/microsoft-is-removing-copilot-branding-from-photos-notepad-and-more-after-promising-to-reduce-unnecessary-copilot-entry-points/) · [TechPowerUp](https://www.techpowerup.com/348135/microsoft-starts-removing-copilot-from-notepad-snipping-tool-and-more-in-windows-11)）。
