## 3. AI 产品更新

### 📌 OpenAI Codex 5/26 大版本：Goals 默认启用、permission profile 引入继承

OpenAI Codex 5/26 上线大版本，goals 默认启用并独立持久化，permission profile 引入 list API 与继承。permission profile 同时支持托管 **requirements.toml**、运行时 refresh 与 Windows sandbox 加固；plugin discovery 引入 marketplace 感知列表、已安装版本、可见 marketplace root 与远程集合；remote-control 改为前台命令、等待 readiness 并报告机器状态，并保留显式 daemon-style start/stop；Chrome 扩展接管已有 tab 后不再创建 tab group，改用 tab icon 表示状态。Codex CLI 与 packaging（installer / npm / runtime）发布管线同步扩容，多项 Windows 与 PowerShell bug 与插件地理可用性问题被修复。下一关键节点：Codex CLI 0.130.0 与 Codex for Chrome 是这次发布串联节点，将在 **6/2-6/4** Code with Claude 大会前接受与 Claude Code、Grok Build CLI 在 SWE-Bench Verified 上的横向对评（详见 2026-05-26.md §3 xAI Grok V9 / Grok Build CLI）。
