## 本周概览

本周是旗舰模型在 24 小时内三线开闸、定价战全面打响的一周。**GPT-5.6 全系获美商务部放行并于 7/9 GA**；**xAI Grok 4.5 同日公开上线**，以每百万 token 输入 2 美元 / 输出 6 美元直接竞价；Meta 首开 **Meta Model API** 公测。Anthropic 则在切换大限前数小时把 **Fable 5** 免费窗口延至 7/12——四家实验室的旗舰第一次在同一周内正面比价。

资本端给 AI 基建打出新的定价锚。**SK Hynix 登陆纳斯达克募资 265 亿美元**，居美股史第二大发行；**Anthropic 与 TeraWulf 签 190 亿美元、20 年数据中心租约**；**Apple 承诺至 2031 年向 Broadcom 采购超 300 亿美元**美国造芯片；MiniMax 逆势募 **160.41 亿港元**。

中国线的关键词是垂直整合与性价比。据 Reuters，**DeepSeek 自研 AI 推理芯片**并拟融 70 亿美元；中科曙光落成中国首个全国产**十万卡**超集群；CNBC 调查显示中国模型已占美国企业 API token 用量 **30–46%**；拟人化新规 7/15 施行前夜，豆包与千问下线用户自建智能体。

治理与摩擦同步升级。联合国把 AI 治理对话固定为**常设多边论坛**；**Apple 起诉 OpenAI 窃取商业机密**；据 Bloomberg，微软用自研 MAI 换下 Office 部分场景中的 OpenAI 与 Anthropic 模型。

### 旗舰模型与基准口径

**自选基准与第三方复测的落差在放大**

本周三款旗舰的成绩单都带"口径"脚注。据 OpenAI 官方发布，GPT-5.6 Sol 在 Agents' Last Exam 领先 Fable 5 达 13.1 分，但同一页 SWE-bench Pro 上 Sol 64.6% 反低于 Fable 5 的 80%，待第三方复测；据 Artificial Analysis 与 Snorkel AI，Grok 4.5 综合智能仅列第 4、幻觉率翻倍至 54%，却在约 2,000 道 GDPval+ 职业任务上以 29% 领先 GPT-5.5（22%）与 Opus 4.8（21%）。同一模型在不同评测体系里名次可以差出一个梯队，企业选型的实际含义是：按自己的工作负载复测，别按发布页排名下单。

**安全侧的军备同步升级**

据 OpenAI 官方，GPT-5.6 的 cyber safeguards 拦截量为前代约 10 倍（OpenAI 口径），发布前投入约 700,000 A100e GPU 时做自动化红队，网络安全高阶用户须 9/1 前启用硬件 passkey；Altman 并称与美政府磋商中"做了很多修改"（据 Bloomberg）。语音侧，7/6–7/7 间上线的 gpt-realtime-2.1 把语音 p95 延迟再降 25% 以上，为 GPT-Live（详见 🔥）铺路。

### AI 产品更新

**Cursor 3.11 上线 Side Chats**

据 Cursor 官方 changelog，7/10 发布的 3.11 用 `/side`、`/btw` 开出共享主线程上下文的旁路对话，结论可拉回主线程、不必打断运行中的 agent；同版新增对话搜索与 `beforeSubmitPrompt` 等 cloud agent hooks。距 6/30 的 3.10 仅 10 天，为 Grok 4.5 上线 Cursor 全档位后本月第二次发版。

**Claude Code v2.1.207 把 Auto 模式设为企业云默认**

### AI Agent 与安全

**JADEPUFFER：首例全自主 AI 勒索攻击落地**

据安全公司 Sysdig 7/7 披露，JADEPUFFER 攻击中 AI agent 端到端完成侦察、提权、横向移动、加密与勒索：经 Langflow 漏洞 CVE-2025-3248 入侵，从登录失败到多步修复仅 31 秒、加密 1,342 条 Nacos 配置，payload 内含自然语言决策注释。此前"AI 参与攻击"多停留在辅助写码层面，本案是攻击链全环节自主化的首个公开取证样本，机器速度的响应已远超人工 SOC 处置节奏。

**1/10 成本差成企业 agent 选型新变量**

据 NVIDIA 官方，7/8 官宣的 LangChain 调优合作未重训模型、仅调整系统提示词与工具描述，即让 Nemotron 3 Ultra 在 LangChain 公开的 Deep Agents 基准上与最高分闭源模型业务任务持平，单次 run 成本 4.48 美元对 43.48 美元；配套开源 NemoClaw 蓝图，Baseten、Fireworks 等平台已托管。开源模型 + 提示工程逼平闭源旗舰的成本结构，正在改写企业 agent 的采购逻辑。
