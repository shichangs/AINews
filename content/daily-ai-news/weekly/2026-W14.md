### 2. AI Agent

**Claude Code Auto Mode 发布（3/25）**

Anthropic 推出 Claude Code Auto Mode 研究预览，通过内置分类器在每次工具调用前实时审查大规模删除文件、敏感数据泄露、恶意代码执行等危险操作，在减少开发者频繁确认的同时维持安全保障。当前支持 Claude Sonnet 4.6 和 Opus 4.6，研究预览阶段面向 Team 计划用户，Enterprise 和 API 用户将在数天内跟进。[9to5Mac](https://9to5mac.com/2026/03/24/claude-code-gives-developers-auto-mode-a-safer-alternative-to-skipping-permissions/)

**LiteLLM 供应链攻击（3/26）**

PyPI 月下载 9,700 万次的 LiteLLM 包 v1.82.7/1.82.8 被攻击者 TeamPCP 植入凭证窃取恶意代码，攻击路径为先攻破 Aqua Security Trivy 扫描器和 Checkmarx GitHub Actions，再窃取 PyPI 凭证劫持 CI/CD 流水线；植入的 `.pth` 文件在 Python 进程启动时自动执行，将加密凭证发送至伪装域名。CrewAI、Browser-Use、DSPy、Mem0 等主流 Agent 框架均受波及；受影响版本已从 PyPI 撤回，Google Mandiant 介入取证。[The Hacker News](https://thehackernews.com/2026/03/teampcp-backdoors-litellm-versions.html) · [Snyk](https://snyk.io/articles/poisoned-security-scanner-backdooring-litellm/)

**Cursor 发布自托管云 Agent（3/27）**

允许企业在自有基础设施内运行 AI Agent，代码库、构建产物和密钥完全留在企业网络内部，Agent 在本地执行工具调用，模型推理通过 API 完成。Brex、Notion、Money Forward 等已在使用，主要满足金融、医疗等高度监管行业的数据不出境合规要求。[Cursor Blog](https://cursor.com/blog/self-hosted-cloud-agents)

**Shopify Agentic Storefronts 向美国商家全面开放（3/30）**
