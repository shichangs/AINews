## 本周概览

本周是出口管制从"全面停服"走向"全球重启"、监管却同步向纵深推进的一周。**Anthropic 旗舰 Fable 5 停服 18 天后于 7/1 全球恢复**，美商务部 6/30 撤销 6/12 管制令，成首个被监管令关停又重启的前沿模型；但监管的手同时伸得更深——据 FT，**白宫与 OpenAI、谷歌、Anthropic 就前沿模型自愿框架进入收尾谈判**，7/2 解封的法庭邮件更曝出**国防部要 Claude 用于自主武器与本土监控、遭 Dario Amodei 拒绝**。

资本向头部集中并刷新纪录。**Crunchbase 上半年报告显示全球风险融资达 5,100 亿美元、已超 2025 全年**，OpenAI 与 Anthropic 两家独占 2,170 亿美元、占 43%；阿布扎比 **MGX 收官 490 亿美元 AI 基金**成史上最大之一；**OpenAI 更提出让美政府持股 5%（约 426 亿美元）**，为其 9 月 IPO 叙事添最大变量。

模型与算力两端同步扩张。**Anthropic 发布 Claude Sonnet 5**，以 2 / 10 美元每百万 token 的介绍价贴近旗舰、成 Free/Pro 默认；Claude 同周在 Microsoft Foundry 转 GA。算力侧，**NVIDIA 推"美国制造"首度在美整链产 Blackwell**、披露 43 州供应链；**字节跳动据报押注巴西建境外最大数据中心（约 390 亿美元）**；Tesla 则在迈阿密上线首日无安全员的 Robotaxi。

### 主要 AI Labs 动态

**Sonnet 5 把旗舰性能压到 agent 价位**

Anthropic 6/30 发布的 Claude Sonnet 5 以 2 / 10 美元每百万 token 的介绍价贴近旗舰 Opus 4.8，据 System Card 在 Terminal-Bench 2.1 达 80.4%、并在 GDPval 上首次微超旗舰（详见 🔥）；据 TechCrunch，这直击企业 Q2 token 预算超支痛点。但迁移非无缝：扩展思考默认常开、temperature 等采样参数被移除、新分词器令同文本多产 1.0–1.35 倍 token，依赖旧参数的生产管线须重建基线。

**Claude 的云与企业侧同步收口**

据 NVIDIA 与微软官方，Claude Opus 4.8、Haiku 4.5 同周在 Microsoft Foundry 转 GA、跑 GB300，Azure 由此成唯一同托 Claude 与 GPT 前沿模型的云（详见 🔥）；配套 7/2 上线的 Claude Enterprise 花费管控——管理员可按角色锁模型、组织级花费达 75% / 90% 时预警——把"token 超支"从增长副作用拉回可管理开支。据 Anthropic，Workato CIO 称把 Claude 接入企业 MCP 后季度营收提升 4%。

**Meta 探索卖闲置算力**

### AI Agent 与企业部署

**主权与气隙成 agent 落地新战场**

据 BusinessWire 与 NVIDIA，Palantir 6/29 联合 NVIDIA 推"智能引擎"，把 Nemotron 开源模型与 Palantir 的 AIP / Ontology / Foundry / Apollo 四件套整合、支持在保密气隙环境部署并保留模型权重所有权，直取 OpenAI 政府版；黄仁勋称开源 AI 是"国家安全与美技术领先的基石"，当日 PLTR 股价涨约 3%。

**安全扫描做成低成本 agent 集群**

据 Cognition，其 7/1 发布的 Devin Security Swarm 用"Agentic MapReduce"跨全库找漏洞、沙箱验证可利用性并自动开修复 PR，官方 eval 在 14 种语言的 50 个真实 GHSA 漏洞上报 72% 召回、单次运行约 90 美元（约为次优工具三分之二成本）；企业能否接受其误报率与自动 PR 质量是采用率首个硬信号。

**能力评估：远程工时仍难被端到端替代**

据 CAIS 与 Scale AI 更新的 Remote Labor Index，Claude Fable 5 以 16.1% 居公开模型首位、约为 Opus 4.8（8.3%）两倍，但即便榜首模型也仅约六分之一真实远程外包项目达可交付标准——agent 距端到端替代远程工时仍远。同周 Accenture 与 ServiceNow 另推 agentic 风控迁移联合托管服务，把遗留系统迁移的成本门槛交给 agent 承接。
