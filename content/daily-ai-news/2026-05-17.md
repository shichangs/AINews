## 2. 主要 AI Labs 动态：安全事件

### 🔥 OpenAI 4 款 macOS 应用须在 6/12 前强制更新

OpenAI 4 款 macOS 应用须在 **6/12** 前更新。ChatGPT Desktop、Codex App、Codex CLI、Atlas 签名证书因 TanStack npm 供应链攻击全部轮换，过期版本将被 Apple 公证体系拦截。

起因是 5/11 TanStack npm 仓库被 "Mini Shai-Hulud" 蠕虫感染，**84 个**恶意包版本横跨 **42 个** `@tanstack/*` 包名被发布。OpenAI 5/15 11:08 UTC 在官方安全博客披露：2 名员工设备触达的有限内部代码仓库被实施"以凭证为目标的外泄活动"，仅有限凭证物料被成功外泄；客户数据、生产系统、对外软件均未受影响。这两台设备恰好"尚未完成"3 月底 Axios 供应链事件后引入的新型软件包管理保护灰度部署。Socket 研究员将 OpenAI 暴露面与 5/12 SAP-related npm 包遭袭事件归为疑似同一 TeamPCP 组织持续行动。**影响范围**：OpenAI 同时启动 iOS、Android 端签名证书审查，预计两周内逐步完成。

（来源：[OpenAI 官方披露](https://openai.com/index/our-response-to-the-tanstack-npm-supply-chain-attack/) · [The Register](https://www.theregister.com/security/2026/05/15/openai-caught-in-tanstack-npm-supply-chain-chaos-after-employee-devices-compromised/5241019) · [The Hacker News](https://thehackernews.com/2026/05/tanstack-supply-chain-attack-hits-two.html)）

## 3. AI Agent 与企业部署

### 🔥 GOV.UK Chat 5/15 全量上线，Claude 接管 gov.uk 80,000 页公民问答

GOV.UK Chat **5/15** 全量上线 GOV.UK App。Anthropic Claude 接管 gov.uk **80,000 页**核心内容的公民问答接口，结束两年半 pilot 阶段——这是 Claude 在海外政府公民服务一线场景已公开披露的部署之一。

软启动期数据：PublicTechnology 5/15 复盘指 GOV.UK Chat 两期 pilot 累计约 **10,000+** 测试用户、覆盖约 **26,000 个**问答，公开测评中 **73%** 评为"有用"、**64%** 表示"满意"。GDS 自报答案准确率约 **90%**，但 LBC 同日报道指出测试中已出现过"误导性"税务答复。这是 Anthropic Claude 在海外政府公民服务一线场景已公开披露的部署之一，与 Anthropic 5/14 同期推进的 PwC 与 Gates Foundation 企业及慈善扩张形成节奏对应。下一关键节点：GDS 预告 6 月将公开发布全量上线首月的准确率与用户满意度追踪数据。
