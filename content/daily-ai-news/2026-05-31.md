## 2. AI Agent 与产品更新

### OpenAI Codex 上线 Windows Computer Use，可操作桌面应用

Codex **5/29** 起可在 Windows 上看、点、输入桌面应用——EEA、英国、瑞士首发不可用。新版 Codex app（**26.527**）让 Codex 在前台操作 Windows 桌面程序完成测试、调试与构建；用户可从 iOS / Android 上的 ChatGPT 或 Mac 上的 Codex 远程续跑同一任务，而 Windows 机器仍是项目文件、shell、应用服务与本地上下文的宿主。同版新增 Codex Profiles（身份 / 历史活动 / 用量 / token 统计）、本地项目线程协调与跨历史线程搜索，并改进应用内浏览器速度与稳定性；企业版默认关闭，需联系账户代表申请早期访问。这把 Codex 从终端编码扩展为对主流企业桌面 OS 的 GUI 自动化，与计算机操作类 agent 正面相遇。下一关键节点：**6/2-6/3** Microsoft Build 的 Windows Agent Framework 是同一桌面战场的竞争面。

（来源：[OpenAI · Codex 更新说明](https://developers.openai.com/codex/changelog/?type=codex-app) · [OpenAI Help · ChatGPT Release Notes](https://help.openai.com/en/articles/6825453-chatgpt-release-notes)）

### 📌 Google 收紧 Gemini 付费配额：失败任务不计、单次请求设上限

Gemini 付费档失败任务不再扣配额、单次请求设用量上限——**5/29** Google 回应投诉。导火索是一名用户 **5/25** 在 X 晒图：从 0% 用量发出一条头像视频提示，**4 分钟**即触顶 **5 小时**限额；Gemini 负责人 Josh Woodward 公开回应"Yikes, let us take a look!"。Google 5 月的订阅重构已把付费档转向按算力计费（每 5 小时刷新、至周上限封顶），本次同时限制单条 Gemini 3.1 Pro 请求可消耗的配额、修复了一两条 Omni 视频即吃光配额的 bug，并把 AI Ultra 的 Omni 生成额度翻倍。这是按算力计费模式上线后的首次压力测试，焦点在 AI Pro 中间档是否仍实用。Google 同日另给 Search 的 AI Overviews 与 AI Mode 加入 Preferred Sources。下一关键节点：付费用户需观察 5 小时窗口在视频等重负载下是否可预期，是 Gemini 订阅口碑的近期检验点。

## 3. 芯片与半导体

### 🔥 NVIDIA 6/1 GTC Taipei 揭首款笔记本 SoC N1X，预热"PC 新纪元"

NVIDIA **6/1** 台北 keynote 首发 **N1X**——其首款 Windows on Arm 笔记本系统级芯片。**5/29** NVIDIA 偕 Microsoft、Arm 贴出台北流行音乐中心坐标预热"PC 新纪元"，黄仁勋此前称"还有一个谁都没告诉过的惊喜新品"，现确认即 N1X：配 **20 核** ARM v9.2 CPU（10 性能 + 10 能效）、Blackwell 架构集显（约对标桌面 **RTX 5070**、6,144 CUDA 核）与完整 CUDA 栈，面向 Windows 笔记本而非数据中心机架。首批整机来自 Dell、Lenovo、Asus、MSI，预计 **2026 假期季**前上市、**2027 年初**扩量。同场 keynote 还将覆盖 Vera Rubin 平台 **2026 H2** 出货排程与 NVL72（号称每瓦推理 **10×**）。下一关键节点：**6/1 11:00**（台北）keynote 是 N1X 规格与 Vera Rubin H2 排程的确认点，后者是 Anthropic 1 GW 级算力交付的时间锚。

（来源：[NVIDIA Blog · GTC Taipei 直播页](https://blogs.nvidia.com/blog/nvidia-gtc-taipei-computex-2026-news/) · [VideoCardz · N1/N1X 预热](https://videocardz.com/newz/nvidia-teases-new-era-of-pc-ahead-of-n1-and-n1x-laptop-chip-announcement) · [Tom's Hardware/TechTimes · N1X 确认](https://www.techtimes.com/articles/317428/20260530/nvidia-arm-laptop-chip-n1x-confirmed-computex-cuda-rtx-5070-gpu-onboard.htm)）

### 📌 Samsung 出货首批 12 层 HBM4E 样品，单日股价 +6%

AI 内存供给竞赛进入认证阶段——Samsung **5/29** 向主要客户出货其称为业界首批的 **12 层 HBM4E** 样品。该样品额定 **14 Gbps**（可扩至 16 Gbps）、单堆带宽 **3.6 TB/s**、容量 **48 GB**（另规划 32 / 64 GB），采用 1c DRAM 工艺与 4nm 逻辑基底芯片，Samsung 称较上代能效 **+16%**、热阻改善 **+14%**；当日股价一度涨 **6%**。这是继 3 月在 NVIDIA GTC 公布 HBM4E、并将 HBM4 量产绑定 Vera Rubin 平台后的下一步。竞争格局上，Micron 在 Q1 已把 HBM4 推入高量产（>11 Gbps、提前一季交付），SK 海力士仍是 2025 年 HBM 市场存量第一。下一关键节点：样品须先过板级测试、热限与供货承诺才能规模部署，6/1 GTC Taipei 的 Vera Rubin H2 排程（见 N1X 段）是这批 HBM4E 的需求时间锚。

（来源：[Samsung Newsroom · HBM4E 样品出货](https://news.samsung.com/global/samsung-electronics-begins-shipment-of-industry-first-hbm4e-samples) · [CNBC · 样品与股价](https://www.cnbc.com/2026/05/29/samsung-hbm4e-chip-samples-ai-memory.html)）

## 4. 中国 AI 产业

### 📌 Tencent WorkBuddy 桌面 agent 出海，押注小模型 + 应用落地

Tencent **5/29** 将 OpenClaw 兼容的桌面办公 agent **WorkBuddy** 出海，押注"低算力 + 应用"对抗阿里、字节。其赌注是：可部署工具与更低算力需求，在把 AI 兴趣转为留存上可与"堆芯片 + 堆模型规模"并重。支撑数据来自 Q1 财报——营收 **RMB 1,965 亿**（+9% YoY）、资本开支 **RMB 319 亿**（+16%）；执行副总裁汤道生称内部 AI 生成代码占比已逾 **50%**（部分团队接近全员）。产品侧，此前发布的 Hy3 Preview（**295B** 参数 MoE）以同参数级的成本效率而非纯规模为卖点，并借微信、QQ 把 agent 嵌入既有用户基；对照阿里仍走硬件 / 规模路线（5/20 推 Zhenwu M890 AI 芯片）。下一关键节点：WorkBuddy 出海是该路线能否在中国以外形成稳定留存的检验，留存数据是判断其有效性的直接信号。
