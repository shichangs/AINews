## 1. 芯片与算力

### 🔥 AMD 向 Anthropic 投资至多 $50 亿，2GW MI450 部署直插 NVIDIA 腹地

MI450 首获前沿实验室多吉瓦级实单，Anthropic 训练算力不再单点押注 NVIDIA。AMD 与 Anthropic 7 月 22 日宣布战略合作：Anthropic 将部署至多 **2 吉瓦（GW）** AMD Instinct **MI450** 系列 GPU，AMD 承诺对 Anthropic 做至多 **$50 亿**战略股权投资、且与部署里程碑挂钩；首个 **1GW** 集群 **2027 上半年**上线。芯片装入 AMD Helios 机架方案（MI455X GPU + EPYC "Venice" CPU），合作建立在 Anthropic 现用 MI355X GPU 的基础上。此约与 Anthropic 近月和 Meta 洽谈约 $100 亿算力租赁、及和 SpaceX、Amazon 的安排并行，显示前沿实验室正把供应链从单一依赖 NVIDIA 分散。下一节点：2027 上半年首个 1GW 交付；AMD Q2 财报 8/5 或披露该单对 Instinct 收入指引的影响。

（来源：[AMD 投资者关系](https://ir.amd.com/news-events/press-releases/detail/1292/amd-and-anthropic-announce-strategic-partnership-to-deploy-up-to-2-gigawatts-of-amd-instinct-mi450-series-gpus) · [CNBC](https://www.cnbc.com/2026/07/22/amd-anthropic-ai-chip-investment.html)）

## 3. AI Agent / 产品更新

### OpenAI 发布企业 Agent 平台 Presence，已接管自家电话支持

上线数周内，OpenAI 自家英文电话支持已由 **Presence** 无人工解决 **75%** 来电。OpenAI 7 月 22 日推出该企业级 Agent 平台：面向客服、外呼销售、高风险内部流程（英文电话支持线为 1-888-GPT-0090），支持实时语音与聊天，把模型推理与策略、护栏、审批、升级规则绑定；每次部署只授予单一任务所需的知识与系统权限。Codex 借 Presence 插件分析生产会话与升级信号、提出可测试审批的改动——在 OpenAI 自身支持线上，该改进循环 **10 天**内把人工交接降 **15 个百分点**。BBVA、SoftBank、IAG 为设计伙伴。行动项：Presence 目前仅经 OpenAI 前置部署工程师面向合格企业限量交付、暂无自助版，企业须联系 OpenAI 客户团队评估。

（来源：[OpenAI 官方博客](https://openai.com/index/introducing-openai-presence/)）

### Cursor Router：Auto 模式改由智能路由，Teams 计划默认开启

Cursor **Auto** 模式即日改由 **Router** 智能路由，含 3 档模式、Teams 默认开启。Cursor 7 月 22 日发布该路由：按任务类型与复杂度逐请求分发模型，提供 Cost / Balance / Intelligence 三档，Balance 与 Intelligence 按所路由模型费率计费；管理员可按团队/分组启用、限制模式、设默认档并允许或屏蔽底层模型；Grok 4.5 被列为价格高效档的必选路由项。覆盖桌面、Web、iOS、CLI 与 SDK。行动项：企业管理员可在 dashboard 启用，并用软/硬约束把团队统一到 Auto。
