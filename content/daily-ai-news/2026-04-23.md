## 2. AI Agent

### 🔥 OpenAI ChatGPT Workspace Agents 上线：Codex 云端 + Slack 原生 + 角色权限
**OpenAI** 于 **4 月 22 日**正式发布 **ChatGPT Workspace Agents**（Research Preview），定位为 **"Codex 云端驱动的团队级共享 Agent"**，**GPTs 正式升级形态**。核心能力：
- **云端 Codex 环境**：Agent 拥有**独立 workspace（文件、代码、工具、记忆）**，**可写/执行代码、调用已连接第三方 App、跨多步执行**，用户断线后**任务仍在云端持续运行**
- **Slack 原生集成**：Agent 可直接在 Slack 频道中**被 @ 唤起、自动分流**；OpenAI 自家产品团队示例 Agent **全天候回答员工问题、附文档链接、必要时开 ticket**
- **定时调度 + 监听触发**：常驻后台 Agent 可**按时段**或**事件**自动接管任务——典型用例：**产品反馈路由、每周报告、潜在客户资质鉴定、合同回复、第三方风险管理**
- **企业治理**：**角色访问控制 (RBAC)、管理员审批、审计日志、监控、集中管理**
- **定价**：**5 月 6 日前全免费**，之后**基于 credit 的计量计费**；**Business / Enterprise / Edu / Teachers** 四档订阅**即时可用**

结合前日 OpenAI 披露 **Codex 周活跃开发者已达 400 万**，以及 **Codex Labs + Chronicle 屏幕感知记忆**，**OpenAI "Codex Stack" 事实上已成 2026 年最完整的 Agent 基础设施**——**对标 Anthropic Computer Use / Google Auto Browse / Nvidia Enterprise Agent Toolkit / Microsoft Copilot Studio**。（来源：[OpenAI Blog](https://openai.com/index/introducing-workspace-agents-in-chatgpt/) · [9to5Mac](https://9to5mac.com/2026/04/22/openai-updates-chatgpt-with-codex-powered-workspace-agents-for-teams/) · [The Decoder](https://the-decoder.com/openai-launches-workspace-agents-that-turn-chatgpt-from-a-chatbot-into-a-team-automation-platform/) · [Prism News](https://www.prismnews.com/news/openai-launches-cloud-workspace-agents-for-chatgpt-business) · [Seeking Alpha](https://seekingalpha.com/news/4578128-openai-brings-workspace-agents-to-chatgpt)）
