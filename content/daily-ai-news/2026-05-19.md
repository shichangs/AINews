## 5. AI 产品：编码 Agent 模型迭代

### 📌 Cursor Composer 2.5 基于 Kimi K2.5 训练，6 月升 cloud agent 默认

Composer 2.5 **Standard 档** $0.50/M 输入 + $2.50/M 输出（**Fast 档** $3/$15），约为 Claude Opus 4.7 / GPT-5.5 的**一成成本**；前 7 天 Cursor 用户使用量按 2 倍计入额度。Cursor 5/18 changelog 公布该模型在 **SWE-Bench Multilingual** 拿到 **79.8%**（Opus 4.7 为 **80.5%**、GPT-5.5 为 **77.8%**），**CursorBench v3.1** 为 **63.2%**；Terminal-Bench 2.0 与 Opus 4.7 持平、落后 GPT-5.5 约 13 分。第三方拆解（The Decoder / Lushbinary）指出 Composer 2.5 在 Moonshot **Kimi K2.5** 开源 checkpoint 基础上训练，合成训练任务数为 Composer 2 的 **25 倍**。下一关键节点：**6 月上旬**前 Cursor 计划把 Composer 2.5 设为 cloud agent 默认模型，预计同步公布 Bugbot 高强度档与 PR Review 流水线接入数据。（来源：[Cursor Changelog · Composer 2.5](https://cursor.com/changelog/composer-2-5) · [The Decoder · 性价比拆解](https://the-decoder.com/cursors-composer-2-5-matches-opus-4-7-and-gpt-5-5-benchmarks-at-a-fraction-of-the-cost/) · [Winbuzzer · 持续任务能力](https://winbuzzer.com/2026/05/18/cursor-releases-composer-25-saying-its-better-at-s-xcxwbn/)）
