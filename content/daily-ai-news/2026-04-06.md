## 1. 主要 AI Labs 动态

### 🔥 Anthropic 发现 Claude 内部存在"情绪向量"
Anthropic 研究团队宣布在 Claude 模型内部发现了类似情绪的信号，研究将其称为**"情绪向量"（emotion vectors）**。这些信号在模型处理复杂伦理困境或情感浓厚的话题时表现最为显著，似乎对 AI 的决策方式产生影响。这是 AI 可解释性领域的重大发现，引发学界对 AI 内在状态的广泛讨论。（来源：[IndexBox](https://www.indexbox.io/blog/anthropic-changes-claude-code-subscription-for-third-party-tools-in-2026/) · [The Currency Analytics](https://thecurrencyanalytics.com/altcoins/anthropic-finds-emotion-like-signals-in-claude-ai-model-251071)）

### 📌 Anthropic 终止 Claude 订阅对第三方工具的覆盖
自 **4 月 4 日中午（太平洋时间）** 起，Anthropic 宣布 Claude 订阅套餐不再涵盖通过外部应用（包括 AI Agent 平台 **OpenClaw**）的使用量。Claude Code 负责人 Boris Cherny 表示："我们的订阅模式并非为这类第三方工具的使用模式所设计。"用户现须通过独立的按量计费或 API Key 访问这些工具，同时将获得一次性等额订阅费用的抵扣额度。该政策预计近期扩展至所有类似第三方工具。（来源：[The Decoder](https://the-decoder.com/anthropic-cuts-off-third-party-tools-like-openclaw-for-claude-subscribers-citing-unsustainable-demand/) · [PYMNTS](https://www.pymnts.com/artificial-intelligence-2/2026/anthropic-says-claude-code-users-will-need-to-pay-more-for-openclaw/)）

### 📌 OpenAI：GPT-5.5 预训练完成，GPT-4o 正式退役
OpenAI 代号 **"Spud"** 的 GPT-5.5 已完成预训练阶段，但仍需经历安全评估、红队测试和阶段性发布准备，短期内不会公开发布。与此同时，GPT-4o 已于 **4 月 3 日** 从所有 ChatGPT 方案中正式退役，标志着 GPT-4 系列全面谢幕。**GPT-5.4 mini** 已向 Free 和 Go 用户开放（通过"+"菜单的 Thinking 功能），并作为 GPT-5.4 Thinking 的限速回退选项。OpenAI 当前月营收达 **20 亿美元**，IPO 筹备仍在推进中。（来源：[AI and News](https://www.aiandnews.com/blog/breaking-ai-news-april-2026/) · [eWeek](https://www.eweek.com/news/openai-laid-out-2026-roadmap-neuron/)）

### 🔥 Google DeepMind 发布 Gemma 4 开源模型家族
Google DeepMind 在 **4 月 2-3 日** 正式发布 **Gemma 4** 家族，基于 Gemini 3 相同研究基础，原生支持**文本、视觉、音频**三模态处理，能力媲美此前仅专有前沿模型才具备的推理水平。Gemma 4 提供四种规格，从边缘端优化小模型到企业级旗舰，并包含一个 **MoE 混合专家**变体，适用于高吞吐量场景。AMD 和高通均为 Gemma 4 提供了零日支持，其中 AMD Instinct MI355X 加速器实现了每秒超 **100 万 tokens** 的推理速度。（来源：[Que.com](https://que.com/gemma-4-redefining-the-frontier-of-open-weights-multimodality/) · [Google Blog](https://blog.google/products/gemini/gemini-3/)）

### 新模型动态：Zhipu AI 发布 GLM-5V-Turbo
智谱 AI 发布 **GLM-5V-Turbo**，这是其首个面向视觉编程任务的多模态基础模型，支持 **200K 上下文窗口**和最大 **128K 输出 tokens**，专为视觉感知的代码生成任务设计。（来源：[DevFlokers](https://www.devflokers.com/blog/ai-news-last-24-hours-april-2026-model-releases-breakthroughs)）

## 2. AI Agent

### 🔥 Linux Foundation 成立 Agentic AI Foundation，MCP 捐入开源生态
**Linux Foundation** 宣布成立 **Agentic AI Foundation（AAIF）**，将 **Model Context Protocol（MCP）**、**goose** 以及 **AGENTS.md** 作为首批项目捐入，使 MCP 从 Anthropic 的专有标准正式演变为行业中立的开源基础设施。MCP 目前已突破 **200 个**服务器实现，97 million 次安装（截至 2026 年 3 月），所有主流云厂商均已接受 MCP 为 AI Agent 连接标准。AAIF 还宣布了 **AGNTCon + MCPCon** 北美和欧洲 2026 年全球活动计划。（来源：[Linux Foundation](https://www.linuxfoundation.org/press/linux-foundation-announces-the-formation-of-the-agentic-ai-foundation) · [PR Newswire](https://www.prnewswire.com/news-releases/linux-foundation-announces-the-formation-of-the-agentic-ai-foundation-aaif-anchored-by-new-project-contributions-including-model-context-protocol-mcp-goose-and-agentsmd-302636897.html)）

### Uber 披露 MCP 规模化实践
Uber 工程团队公开介绍了如何使用 **MCP** 规模化其工程服务和 AI Agent，Uber Agentic AI 负责人表示："MCP 不仅重要，它真的让 AI 在 Uber 变得有用。"（来源：[AI Business](https://aibusiness.com/agentic-ai/mcp-alive-faces-challenges)）

### Domo 发布 AI Agent Builder 和 MCP Server
Domo 推出新的 AI 编排框架，包括 **AI Agent Builder**、**AI Toolkits**、集中式 **AI Library** 和 **Domo MCP Server**，旨在帮助企业从 AI 试点快速过渡到协调化系统，支持将企业数据直接连接至外部 AI 平台。（来源：[Demand Gen Report](https://www.demandgenreport.com/industry-news/news-brief/domo-launches-ai-agent-builder-mcp-server-to-connect-enterprise-data-to-ai-ecosystem/52332/)）

## 3. AI 产品更新

### Cursor 3 发布：面向 Agent 工作流的统一开发工作空间
**Cursor 3** 正式发布，带来全新界面设计，核心亮点包括：**多 repo 布局**支持、本地与云端 Agent 无缝切换，以及统一的 Agent 侧边栏（可汇总来自 Mobile、Web、Slack、GitHub、Linear 等渠道发起的任务）。云端 Agent 可生成演示截图供开发者验证。此外，Cursor 引入 **实时强化学习（real-time RL）** 用于 Composer 训练，每隔五小时更新一次检查点，显著降低延迟。（来源：[Releasebot](https://releasebot.io/updates/cursor)）

### Claude Code 更新：无闪烁渲染、named subagents 等多项改进
Claude Code 最新版本新增：**CLAUDE_CODE_NO_FLICKER** 环境变量（支持无闪烁 alt-screen 渲染）、**@ 提及的 named subagents** 显示、**PermissionDenied** hook（在自动模式分类拒绝后触发），以及更广泛的 PowerShell 支持。性能方面，Write 工具对大型文件的 diff 计算速度提升 **60%**，同时修复了长会话稳定性、macOS Keychain 读取失败、IME 输入法光标追踪等多项问题。（来源：[Claude Code Docs](https://code.claude.com/docs/en/changelog) · [Releasebot](https://releasebot.io/updates/anthropic/claude-code)）

### ChatGPT 扩展第三方应用集成
OpenAI 正在 ChatGPT 中向用户推出更新版 **Box、Notion、Linear、Dropbox** 应用，新增写入操作等功能（视平台支持情况而定），进一步拓展 ChatGPT 的企业工具生态。（来源：[Releasebot](https://releasebot.io/updates/openai/chatgpt)）

### PrismML 发布全球首个商业可用 1-bit 大模型
**PrismML** 正式从隐身模式中现身，发布 **Bonsai** 系列 1-bit LLM，其中 8B 模型仅需 **1.15GB 内存**，在 M4 Pro 上运行速度达 **131 tokens/秒**，是 edge 端推理领域的重要突破。（来源：[DevFlokers](https://www.devflokers.com/blog/ai-news-last-24-hours-april-2026-model-releases-breakthroughs)）

## 10. 政策与监管

### 美国联邦 vs. 州法博弈：特朗普行政令挑战州 AI 法律
特朗普政府已签署行政令，要求司法部长挑战与"最低负担国家政策框架"相冲突的州级 AI 法律，涉及加利福尼亚《AI 透明度法》、科罗拉多《AI 法案》、德克萨斯《负责任 AI 治理法》等。美国仍缺乏联邦层面的综合 AI 立法，监管格局以各州法律为主导。（来源：[OneTrust](https://www.onetrust.com/blog/where-ai-regulation-is-heading-in-2026-a-global-outlook/) · [Cimplifi](https://www.cimplifi.com/resources/the-ai-regulation-landscape-for-2026-what-legal-and-compliance-leaders-need-to-know/)）

### 欧盟 AI 法案部分生效，正讨论放宽高风险 AI 义务
**欧盟 AI 法案**的通用用途 AI 和禁止性 AI 实践义务已正式生效，但欧盟立法者正考虑通过"数字综合法规"延缓部分高风险 AI 系统的合规时间表，并放宽 AI 训练数据的个人信息限制。（来源：[IAPP](https://iapp.org/news/a/global-ai-law-and-policy-tracker-highlights-and-takeaways) · [Meta Intelligence](https://www.meta-intelligence.tech/en/insight-ai-regulations-global)）

### 中国推进 AI 生成内容强制标注措施
