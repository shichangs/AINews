## 1. 主要 AI Labs 动态

### 🔥 Anthropic 5/6 在 SF 召开 Code with Claude 旗舰开发者大会：dreaming / outcomes / multi-agent orchestration 公测 + Code Review 上线

Anthropic 于 5 月 6 日在 San Francisco SVN West 举办 **Code with Claude 2026** 旗舰开发者大会，伦敦 5/19、东京 6/10 接力。会上重点发布 **Claude Managed Agents** 的三项新能力：

- **dreaming**：作为研究预览推出，定期扫描 Agent 会话与记忆库，提炼 patterns、整理记忆，让同一 Agent 在多次会话间持续改进。法律 AI 公司 Harvey 在长文起草与文档创建任务里测试 dreaming，完成率较此前提升约 **6 倍**。
- **outcomes**：进入公测，让开发者用 rubric 描述"成功"，由独立的 grader 在自己的上下文里打分，避免被被评估 Agent 的推理结果污染。
- **multi-agent orchestration**：进入公测，与 webhooks 一同向 Managed Agents 开发者开放，可编排专门 Agent 协同完成复杂任务。

**Claude Code** 同日推出 **Code Review**——Anthropic 称已在公司内部全员使用，目的是把开发者从 PR review 解放出来。Anthropic 产品负责人 Ami Vora、Claude Code 负责人 Boris Cherny、Claude API 与 SDK 产品负责人 Angela Jiang 主讲。Simon Willison 在现场 live blog 中确认大会同步以全程 livestream 对外开放，社区因报名超额，Anthropic 增设面向独立开发者与早期创始人的第二日。

*披露：本报告由 Claude 撰写，Anthropic 为 Claude 开发方。dreaming / outcomes / orchestration 由 VentureBeat、9to5Mac 等非 Anthropic 来源交叉验证。*

### 🔥 Anthropic 与 SpaceX 签 Colossus 1 全调拨：300MW / 22 万 GPU + Claude 限额翻倍

Anthropic 5 月 6 日宣布与 SpaceX 达成算力协议，调用 Memphis 的 Colossus 1 数据中心全部容量——超过 **300 MW** 加 **220,000 块 NVIDIA GPU**，将在一个月内并入 Claude 的服务栈。Colossus 1 此前是 xAI 已建成的训练超算，本次以 SpaceX 名义对外开放。Anthropic 官方同篇 newsroom 文章同步宣布：

- Claude Code 在 Pro / Max / Team / 席位制 Enterprise 的 **5 小时限额翻倍**，Pro / Max 的峰时降速取消，周限额不变；
- Claude Opus 模型的 API rate limits 大幅提升，Tier 1 的最大输入 / 输出 tokens-per-minute 分别提升 **15 倍 / 9 倍**。

双方在公告中表达"探索多 GW 在轨 AI 算力"的合作意向，但未给具体时间表。CNBC、Axios、Inc. 同日均把该协议放在 SpaceX 4 月 1 日机密递交 SEC 招股、目标估值 **$1.75–2 万亿** 的背景下讨论：把 Anthropic 写进招股资料，对 SpaceX 描述 AI 基础设施现金流是一条新增叙事线。Musk 当日在 X 上称过去一周与 Anthropic 高层多次接触后"印象深刻"——这与他自己起诉 OpenAI 的庭审同步进行，构成戏剧化反差。

按公开披露汇总，本协议接续 Anthropic 已宣布的 Amazon ~5 GW、Google / Broadcom 5 GW、Microsoft / NVIDIA **$300 亿** Azure 容量、以及 Fluidstack **$500 亿** 美国基础设施承诺，是 Claude 训练 / 推理算力栈的新增独立来源。

*披露：本报告由 Claude 撰写，Anthropic 为 Claude 开发方。SpaceX 算力数字与 SEC 招股估值由 CNBC / Axios 等非 Anthropic 来源交叉验证。*

（来源：[Anthropic Newsroom](https://www.anthropic.com/news/higher-limits-spacex) · [CNBC](https://www.cnbc.com/2026/05/06/anthropic-spacex-data-center-capacity.html) · [Axios](https://www.axios.com/2026/05/06/anthropic-spacex-elon-musk-compute) · [Inc.](https://www.inc.com/ben-sherry/anthropic-and-spacex-just-announced-a-colossal-deal-to-supercharge-claude-ai/91341165)）

## 2. AI Agent

### Cursor 5/6 发布 Composer 自举研究帖：Anysphere 公布 autoinstall 模型训练栈细节

Anysphere 旗下 **Cursor** 5 月 6 日在官方博客发表署名研究帖 *Bootstrapping Composer with autoinstall*，由 Shomil、Joshua 与 Andrew 共同执笔。Composer 是 Cursor 自研、面向代码生成做优化的推理模型，autoinstall 是公司用来给 Composer 持续训练 / 收集分布内问题的内部流水线。帖子描述了"用现有 Composer 给下一代 Composer 生成训练样本 + 自动验证 + 自动安装到调度集群"的闭环，目的是不依赖外部数据采集即可推进迭代。

Cursor 此前已对外披露 ARR 约 **$20 亿**、估值约 **$500 亿**（5 月初 The Next Web 报道，与本帖无直接关联）。这一研究帖与 5/6 同日 Anthropic 在 Code with Claude 公布的 Claude Code Review、Multi-Agent Orchestration 公测一起，把"代码 Agent 自我演进 / 自我训练"主题集中在同一日内。

CNBC 5 月 6 日刊出对香港上市的人工智能公司 **SenseTime（商汤）** 联合创始人、首席科学家 **林达华** 的专访，主题是"被美方制裁的中国厂商如何在全球 AI 竞赛中靠成本效率立足"。林达华向 CNBC 表示，公司主力多模态模型 **SenseNova U1** 在生成 / 理解一体化设计下成本约为 OpenAI ChatGPT Images 2.0 的 **1/10**，并称会沿 DeepSeek 之前打出的"低成本 + 高性能"路径继续向海外市场推。

## 5. 芯片与半导体

### 🔥 AMD Q1 FY26 财报后 5/6 美股盘前 / 盘中 +15%：数据中心 +57% 同比、MI450 / Helios H2 起量

AMD 于 5 月 5 日盘后披露 **Q1 FY26** 财报，5/6 周三美股交易日内消化：

- 营收 **$102.5 亿**（同比 +38%），市场预期 $98.9 亿；
- 调整后 EPS **$1.37**，市场预期 $1.29；
- 数据中心营收 **$58 亿**（同比 +57%），由 EPYC 服务器 CPU 与 Instinct GPU 驱动；CEO Lisa Su 在通稿与 5/6 CNBC 专访中称数据中心已是公司"营收与利润首要驱动力"。
- 业绩展望：MI450 系列与 Helios 机柜系统在 2026 H2 起量，与 Meta 此前披露的 **6 GW** 多年 GPU 合同对应的批量交付亦在 H2 启动。

5/6 美股开盘后 AMD 一度涨逾 **15%**，盘中 Lisa Su 在 CNBC 节目中重申"AI 服务器 CPU 与加速器在 2026 是收入主线"，并把对未来几年 AI 服务器 CPU + GPU 合计 TAM 上修至 **$1,200 亿+**（公司层面口径、未列明年化窗口的具体年限）。这一表态与同期 NVIDIA 5/20 Q1 FY27 财报、Intel-Apple 制造合同传闻一起，构成 5 月初"Wall Street AI 芯片注意力从 NVIDIA 部分外溢到 AMD / Intel / Micron"叙事的关键数据点。

### EU AI Act Digital Omnibus 第三轮 trilogue 5/6 晚 在布鲁塞尔开议

欧洲议会、欧盟理事会、欧盟委员会三方就 **Digital Omnibus on AI** 的第三轮 trilogue 于 5 月 6 日傍晚在布鲁塞尔开议。该轮谈判承接 **4/28** 第二轮"12 小时无果而散"的僵局，焦点是 Annex III 独立高风险系统与 Annex I 嵌入式 AI 的合规期是否分别推迟至 2027/12/2 与 2028/8/2，以及 Article 5 是否新增对"未成年性虐待 / 非自愿性"AI 内容的禁止条款。Bird & Bird、IAPP 等法律观察方在 5/6 当天预判"延期 + 新增禁令"双向妥协是最可能落点。会议持续至 5/7 凌晨——具体结果与文本条款见 2026-05-07.md 跟进。
