## 2. AI Agent

### TikTok World '26 发布 Ads MCP Server：Claude / ChatGPT 可直接接管 TikTok 广告投放

TikTok 5/12 在 TikTok World '26 发布 **TikTok Ads MCP Server**，让外部 AI agent（Claude、ChatGPT、Gemini 等）通过 MCP 协议直接规划、发起、优化 TikTok 广告投放——创意上传、出价调整、预算挪移、定向调整等手工操作可由 agent 接管。

Google Ads MCP、Meta Ads MCP、Amazon Ads MCP 已先后落地，TikTok 是第三家主流广告平台开放 MCP，标志 MCP 协议从开发者工具栈向广告 / 电商等高 ARPU 应用层扩张。

（来源：[TikTok Newsroom](https://newsroom.tiktok.com/tiktok-world-26-turning-discovery-into-business-growth-with-ai-powered-innovations-vertical-experiences-and-high-impact-brand-solutions?lang=en) · [Digiday](https://digiday.com/marketing/tiktok-launches-mcp-server-to-let-ai-agents-run-campaigns/) · [Social Media Today](https://www.socialmediatoday.com/news/major-ad-tool-announcements-from-tiktok-world-2026/820197/)）

## 3. AI 产品更新

### 🔥 OpenAI 上线 Daybreak 企业 AI 安全平台，三档 GPT-5.5、8 大安全厂商首批客户

OpenAI 5/12 发布 **Daybreak**：把 OpenAI 模型与 Codex Security 串联，用于"识别 / 验证 / 优先级排序"软件漏洞、自动生成与测试 patch、留下 audit-ready 证据。

模型三档：**GPT-5.5**（通用 / 含标准保护）、**GPT-5.5 with Trusted Access for Cyber**（已验证企业内防御工作流）、**GPT-5.5-Cyber**（更 permissive，授权 red team / 渗透测试 / 受控验证）。首批 8 家安全厂商：Akamai、Cisco、Cloudflare、CrowdStrike、Fortinet、Oracle、Palo Alto Networks、Zscaler，访问目前为邀请制。

（来源：[Daybreak 官网](https://openai.com/daybreak/) · [The Hacker News](https://thehackernews.com/2026/05/openai-launches-daybreak-for-ai-powered.html) · [Infosecurity Magazine](https://www.infosecurity-magazine.com/news/openai-daybreak-secure-by-design/)）

### Anthropic 发布 Claude for Legal，与 Thomson Reuters CoCounsel Legal 完成 MCP 集成

Anthropic 5/12 发布 **Claude for Legal**：12 个法律执业插件（公司、监管、雇佣等）+ 20+ MCP 连接器，可接入律所现有法律科技栈。同日 Thomson Reuters 宣布将 Claude 通过 MCP 接入 **CoCounsel Legal**，并称下一代 CoCounsel 基于 **Claude Agent SDK** 重写——支持自动规划、工具选择、检索引证、流程中重规划。

CoCounsel 覆盖 **107 国 100 万专业人员**，背后承载 Westlaw、Practical Law、KeyCite 共 175 年法律内容、约 **19 亿文档与 14 亿 KeyCite 信号**；客户数据不用于训练第三方模型。

（来源：[Thomson Reuters](https://www.thomsonreuters.com/en/press-releases/2026/may/thomson-reuters-and-anthropic-expand-partnership-to-connect-claude-with-cocounsel-legal) · [Artificial Lawyer](https://www.artificiallawyer.com/2026/05/12/claude-for-legal-launches-may-reshape-the-legal-tech-world/) · [Bloomberg Law](https://news.bloomberglaw.com/legal-ops-and-tech/anthropic-pushes-deeper-into-legal-work-with-claude-updates)）

### Windows 11 May 2026 Patch Tuesday：AI Agent 任务栏监控首次落地

Microsoft 5/12 发布 Windows 11 May 2026 Patch Tuesday（KB5089549 / KB5087420，覆盖 23H2 / 24H2 / 25H2），修复 **120 项漏洞**。AI 相关新增：**AI agent API** 让 agent 在任务栏作为"应用"被监控，首批接入 Microsoft 365 Copilot Researcher agent，agent 生成报告时在任务栏实时显示进度与完成通知；同步上线 Xbox mode。这是 Windows 首次把 agent 监控作为系统级 UI 暴露。

（来源：[Neowin](https://www.neowin.net/news/windows-11-kb5089549-patch-tuesday-brings-xbox-mode-file-explorer-improvements-and-more/) · [Windows Forum](https://windowsforum.com/threads/windows-11-may-2026-patch-tuesday-xbox-mode-ai-taskbar-agents-and-driver-trust.417875/)）

## 4. 机器人与物理 AI

### Hello Robot 发布 Stretch 4：开源、传感器丰富、起价 $29,950 的人机协作平台

Hello Robot 5/12 发布 **Stretch 4** 开源机器人平台，起价 **$29,950**，定位"通用物理 AI"研究与应用——伸缩臂、全向底盘，传感器含 2 颗半球 3D LiDAR、3 颗高分辨率摄像头、6 颗激光线，向 Waymo 的"sensor-rich"自动驾驶安全哲学靠拢。

落地用例：与四肢瘫痪等重度行动障碍人群试验，用手机界面驱动机器人完成拿水、关窗帘、自助进食；新电源系统支持 8 小时连续运行 + 自动充电桩。Stretch 4 不直接对标 Optimus、Figure 03、Digit 量产人形整机，瞄准研究与人机协作细分赛道。

（来源：[Robotics Tomorrow](https://www.roboticstomorrow.com/news/2026/05/12/hello-robot-unveils-stretch-4-a-simply-useful-robot-that-puts-people-first/26547/) · [IEEE Spectrum](https://spectrum.ieee.org/stretch-4-home-robot) · [The Robot Report](https://www.therobotreport.com/hello-robots-latest-stretch-4-is-bigger-faster-and-stronger-than-previous-versions/)）
