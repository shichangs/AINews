## 3. 政策与监管：欧盟主权 AI

### 欧盟选定 Domyn 领衔的 EUROPA 财团造 400B+ 开源模型

欧盟把主权前沿模型押给 Domyn 领衔的 **EUROPA** 财团，独享至多 **2.5%** EuroHPC 算力一年。欧盟委员会 6/19 公布 Frontier AI Grand Challenge 评选结果，由意大利公司 **Domyn** 牵头的 EUROPA 财团胜出；该挑战赛 2 月启动，要求参赛模型参数过 **400B**、开源开放，并覆盖全部 **24** 种欧盟官方语言。中选方将获得一台或多台 AI 优化型 EuroHPC 超算上限 2.5% 的算力配额、为期一年，用于把模型训到全球前沿水平。下一关键节点：EUROPA 首个权重发布时点，及该模型能否缩小欧盟与美中旗舰的代差。

（来源：[European Commission](https://digital-strategy.ec.europa.eu/en/news/commission-selects-europa-consortium-winner-frontier-ai-grande-challenge-project-build-european) · [IEU Monitoring](https://ieu-monitoring.com/editorial/eu-commission-picks-europa-consortium-led-by-domyn-to-build-open-frontier-ai-model/1243623)）

## 4. AI Agent 与产品

### Mistral 发布统一 agent Vibe，同推工业物理 AI 与 10 MW 数据中心

**Mistral** 6/19 同步三连发，把单一 agent、工业物理 AI 与自建算力一次铺开。新发布的 **Vibe** 是"一个 agent 干工作 + 写代码"：**Work Mode** 跑长流程（清理邮件 / 日历、深度研究、起草交付物），**Code Mode** 做远程编码与提交 PR，配套 VS Code 插件与 CLI 更新。企业平台另接入 Emmi AI 的物理 AI，主打制造、航空航天、能源、半导体的仿真与实时数字孪生。算力侧，位于 Essonne 的 Les Ulis **10 MW** 推理数据中心定于 Q3 2026 投用，以自控产能对冲供应链风险。下一关键节点：Vibe 的长流程 agent 与 Cursor、Claude Code、Codex 的企业渗透之争，及 Les Ulis Q3 投产进度。
